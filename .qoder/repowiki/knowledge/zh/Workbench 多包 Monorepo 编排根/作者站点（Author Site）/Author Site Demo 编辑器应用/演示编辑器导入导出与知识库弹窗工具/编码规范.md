- 所有弹窗均为受控组件，通过 props 暴露 `open`/`onOpenChange`，内部状态仅用于本地 UI（如 loading、content），不持有持久数据。
- 异步操作统一包裹 try/catch/finally，成功时通过 `useToast().toast({ title, description })` 反馈，失败时使用 `variant: 'destructive'` 的错误 toast。
- 按钮在异步期间禁用并显示 `Loader2` 旋转图标，提交前校验必填字段后再设置 loading 状态。
- 对外 API 调用采用相对路径 `/api/...` 配合 `URLSearchParams` 拼接查询参数（projectId/sessionId/workingDir），而非集中式请求封装。
