- 对外暴露的导出均为纯函数（如 `initializeDatabase`、`generateSchemaFromCode`、`validateNoSchemaConflict`），避免在模块作用域维护可变状态。
- Schema 相关函数对非法输入采用“返回空/默认值而非抛错”的防御式策略（`extractSchemaDefaults`、`parseSchemaString`、`validateNoSchemaConflict` 均 catch 后回退）。
- TypeScript 类型定义与实现放在同一文件中，以 `export interface` 形式同时作为文档契约（`SchemaProperty`、`GeneratedSchema`、`JsonSchema`、`SchemaConflictResult`）。
