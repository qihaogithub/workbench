- 对外暴露的公共 API 均为纯函数，接收不可变输入并返回新对象，不修改传入的 scene/node。
- 可变集合（Set/Map）优先于数组+includes 进行 O(1) 查找，用于 ID 去重、子节点引用校验和移动节点过滤。
- 数值字段统一通过 `isFinitePositive` / `isFiniteNonNegative` / `styleNumber` 守卫，避免 NaN/Infinity 污染渲染。
- 所有注入到 HTML/SVG 的属性与文本都经过 `escapeHtml` / `escapeAttr` 转义后再拼接，防止 XSS。
- 补丁操作采用 `op` 字面量联合类型 + switch/if-chain 分发，每个分支先构造候选结果再经 `commitNodesIfValid` 验证后提交。
- 枚举值同时维护 TS 字面量类型与同名的 `SKETCH_SCENE_*_VALUES` Set 白名单，校验时以 Set.has 兜底。
