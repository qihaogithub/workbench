模块以 `src/index.ts` 作为统一导出入口，按职责划分为若干子层：
- 类型契约层：`types.ts` 集中定义 `PreviewPanelProps`、`CanvasState`、`CanvasPageLayout`、`CanvasToolMode` 等核心接口；`iframe-types.ts` 定义父子 iframe 间 postMessage 协议（`IframeInMessageType` / `IframeOutMessageType`），由 `types.ts` 重新导出形成对外边界。
- 运行时调度层：`canvas-render-scheduler.ts` 根据视口中心距离、最近访问时间与预算上限计算每页的渲染模式（`prototype/screenshot/iframe/loading/sleeping-iframe`）；`preview-runtime-pool.ts` 在其基础上包装出 `computePreviewRuntimePoolPlan`，供上层决定哪些页面保留 runtime。
- 预览内核层：`PreviewPanel.tsx` 是核心容器，负责编译缓存（`compile-cache.ts`）、CDN 资源指纹与预取（`preview-resource-cache.ts`）、缩放（`preview-scale.ts`）、配置注入（`preview-config-utils.ts`）、静态快照提取与 DOM 清洗，并通过 `IframePreviewFrame.tsx` + `iframe-template.ts` 生成沙箱 HTML 与跨域通信。
- 画布交互层：`canvas-kernel.ts` 提供坐标转换、指针路由、图层归一化、文本节点摘要等纯函数；`canvas-layout.ts` 处理布局尺寸解析与持久化 key；`CanvasViewport.tsx` / `CanvasPageItem.tsx` / `CanvasFreeNodeItem.tsx` / `CanvasDocumentContent.tsx` / `CanvasToolbar.tsx` 组合成可拖拽、标注、文档浏览的画布界面。
- 配置与编辑器层：`ConfigForm.tsx` + `config-categories.ts` + `validator.ts` 基于 @rjsf 生成动态表单，`widgets.tsx` 注册 ColorPicker/FileUpload/RichText 自定义 widget；`RichTextEditor.tsx` 基于 TipTap 实现富文本编辑；`NoteButton.tsx` / `NoteDialog.tsx` / `NotePreview.tsx` 提供注释能力。
- 依赖方向单向向下：UI 组件 → 调度/内核纯函数 → 类型与 iframe 协议；不反向依赖上层应用状态。
