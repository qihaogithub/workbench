模块为单包 `@workbench/agent-client`，仅含三个源文件：
- `src/types.ts`：纯类型定义层，集中声明与后端契约一致的 DTO（`AgentInfo`、`FilesResponse`、`WorkspaceInfo`、`SendMessageOptions`、`StreamEvent` 等）以及统一响应包装 `ApiResponse<T>`。
- `src/client.ts`：实现两个对外类——`AgentClient`（基于原生 `fetch` 的 REST 客户端，提供 `/api/agent/*`、`/api/sessions`、`/health`、`/api/tools/capabilities` 等端点；通过 `stream()` 将 `http(s)` 前缀替换为 `ws(s)` 构造 `AgentStream`）和 `AgentStream`（基于原生 `WebSocket` 的事件总线，内置指数退避重连、`send/cancel/ping/close` 及 `on/off` 事件订阅）。
- `src/index.ts`：作为包入口，仅 re-export `AgentClient`、`AgentStream` 及若干关键类型，不暴露内部实现。
依赖方向单向：`client.ts` → `types.ts`，`index.ts` 聚合二者；无第三方运行时依赖，构建产物由 `tsc` 输出到 `dist/`，并通过 `exports` 字段同时提供 typescript 源码路径与编译产物。
