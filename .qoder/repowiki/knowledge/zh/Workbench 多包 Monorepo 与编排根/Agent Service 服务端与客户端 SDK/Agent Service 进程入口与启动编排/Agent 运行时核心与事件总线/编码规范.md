- 全局单例通过模块级私有变量 + `getXxx()` 函数懒初始化，避免在模块顶层直接 new 实例。
- 事件采用 discriminated union：每个事件对象带字面量 `type` 字段（如 `stream`、`tool_call`、`finish`），并由 `EventType` 联合类型约束，handler 通过 `on(eventType, handler)` 按字符串订阅。
- Agent 状态变更一律经 `BaseAgent.setStatus` 内部方法，同时更新 `lastActivityAt` 并自动 emit `status` 事件，禁止外部直接写 `_status`。
- 可选能力通过 `?.` 探测（如 `backend.start`、`backend.setModel`、`backend.updateConfig`），不支持时抛错或 warn 日志，保持对弱实现的兼容。
- 配置变更走 `updateConfig(Partial<AgentConfig>)` 并在检测到变化后 emit `config_updated` 事件，由 Manager 层比较 JSON 序列化结果判断是否需要重建 Agent。
