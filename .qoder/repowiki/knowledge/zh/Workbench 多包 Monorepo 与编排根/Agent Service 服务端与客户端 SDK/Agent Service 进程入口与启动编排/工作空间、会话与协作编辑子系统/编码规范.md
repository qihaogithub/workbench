- 全局单例通过模块级变量 + 工厂函数导出（如 `workspaceManager`、`collabRoomManager`、`getSessionStore()`），并在 dispose/destroy 中显式释放定时器与资源。
- 对外部不可信输入一律先走 `validateSession` / `validateWorkspaceSession` 校验，失败后以闭包码（如 1008/1013）或抛错拒绝访问。
- 异步副作用（文件保存、git 操作）统一用 try/catch 包裹并记录结构化日志，错误信息包含上下文键（roomKey、workingDir、filePath）。
- 可配置参数优先从环境变量读取并提供默认常量（如 `COLLAB_SAVE_DEBOUNCE_MS`、`COLLAB_ROOM_IDLE_TTL_MS`、`SESSION_EXPIRY_MS`），便于部署期调优。
