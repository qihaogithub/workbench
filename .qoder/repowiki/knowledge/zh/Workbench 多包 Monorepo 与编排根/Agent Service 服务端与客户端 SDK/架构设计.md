本模块由两个子包构成：
- packages/agent-service：单进程 Fastify 服务，按固定装配顺序启动（Fastify → 后端供应商注册 → 工作区权限恢复 → Agent 工厂 → 路由），通过 routes/ 暴露 REST + WebSocket 接口，核心由 core/agent-manager.ts 统一调度 backends/ 下的各 LLM 供应商实现，并依赖 session/、workspace/、collab/ 等横向能力。
- packages/agent-client：纯 TypeScript SDK，封装与上述同一组 REST/WS 端点的调用，作为 agent-service 的对外消费面。
两端的契约集中在 types.ts 与 routes/ 中定义，client SDK 仅依赖该契约而不感知服务端内部结构，形成“服务端实现 / 客户端契约”的双边边界。
