- Node 脚本统一采用 ESM import 语法与 `.mjs` 后缀，并通过 `process.cwd()` 而非硬编码绝对路径定位仓库根。
- 错误/告警收集模式：声明 `errors = []` / `warnings = []`，在末尾分别打印并依据 `errors.length > 0` 返回非零退出码。
- CLI 参数解析优先读取环境变量（如 `SYNC_STATUS_*`、`HEADLESS`、`E2E_USER`），再以 `--flag <value>` 覆盖，未知选项抛出错误并打印 usage。
- 外部命令执行一律通过 `child_process.spawnSync` 或 `spawn`，并将 `cwd` 设为仓库根、`stdio: 'inherit'` 透传输出，失败时抛错或记录 error。
- Bash 部署脚本以 `set -e` 开头，通过 `case` 白名单校验环境变量（如 `DEPLOY_SERVICES`、`DEPLOY_SYNC_MODE`、`DEPLOY_BUILD_MODE`），非法值直接 exit 1。
