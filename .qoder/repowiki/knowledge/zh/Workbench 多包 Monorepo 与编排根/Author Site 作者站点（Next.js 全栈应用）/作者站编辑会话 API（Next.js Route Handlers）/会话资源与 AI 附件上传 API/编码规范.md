- 每个 route.ts 在入口处先 `getAuthCookie()` + `verifyToken()`，未登录返回 `UNAUTHORIZED`，过期返回 `登录已过期`，形成统一的鉴权守卫模式。
- 对外错误统一用 `createApiError(code, message)` 包裹，成功响应用 `createApiSuccess(data)`，避免直接拼接 JSON。
- 文件类型白名单采用双检：同时校验扩展名集合与 MIME type，并对 `.svga` 放宽 MIME 为空或 `application/octet-stream` 的情况。
- 远程 URL 下载前必须经过 `assertPublicHttpUrl` 进行协议白名单、localhost 阻断、DNS 解析与私网 IP 过滤，再发起 http/https 请求。
- 工作空间写操作遵循 `stageWorkspaceBinary` + `commitWorkspaceMutation` 的两阶段提交模式，put_binary 操作附带 stagingId/hash/size 与 expectedHash 乐观锁。
