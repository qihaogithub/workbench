- 所有 REST 请求返回值统一包裹为 `{ success: boolean; data?: T; error?: { code; message } }` 形式的 `ApiResponse<T>`，并在成功时调用 `mutate('/api/...')` 使 SWR 缓存失效。
- Mock 实现 (`mock-api.ts`) 与真实 API (`api.ts`) 保持相同的函数签名和 `ApiResponse<T>` 返回结构，便于在开发/测试中直接替换。
- 文件系统路径一律通过 `getProjectsDir()`、`getTemplatePath(id)`、`getSessionPath(...)` 等 accessor 组合 `path.join` 构造，禁止硬编码字符串拼接。
- Workspace 元数据集中写入 `workspace-tree.json`，读取时若缺失则自动执行 `migrateLegacyToTree` 从旧 `.folders.json` + `.demo.json` 迁移并回写新格式。
- Yjs 协同状态通过 awareness 的 `presence` 字段广播，包含 `userId`、`username`、`color`、`activePageId`、`resourcePath`、`lastActiveAt`，并用 `arePresenceListsEqual` 做浅比较避免重复渲染。
