- 对外 API 统一采用 discriminated union 结果类型 `{ ok: true; ... } | { ok: false; error: string }`，调用方通过 `ok` 字段判断成功与否。
- 配置目标描述使用 `kind: 'text' | 'image' | 'color'` 三态枚举，并在 `colorProperty?: 'color' | 'backgroundColor' | 'borderColor'` 上附加具体样式属性。
- JSON Schema 生成遵循固定模板：type=object + properties + required，并通过 `ui:options.category` 标注分组，format 用 image/color 标识特殊字段。
- 字段 key 校验统一要求 `/^[A-Za-z_][A-Za-z0-9_]*$/` 且禁止以双下划线开头，标题为空或 defaultValue 非字符串时直接拒绝。
- DOM 节点稳定标识优先使用 `data-ow-id` 属性，回退到 `domPath` 选择器，并通过 `ensureStableNodeId` 自动补全。
