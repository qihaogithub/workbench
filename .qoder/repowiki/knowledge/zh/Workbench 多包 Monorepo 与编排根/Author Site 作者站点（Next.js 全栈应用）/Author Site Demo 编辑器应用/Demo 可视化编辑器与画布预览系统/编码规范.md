- 所有交互逻辑封装在自定义 hook 中（`use*` 命名），页面只负责组合与 props 透传，避免在 page 内写业务分支。
- 可撤销操作统一通过 `useCommandHistory.recordCommand({ label, undo, redo })` 注册，键盘快捷键由 `bindKeyboardShortcuts` 集中绑定，忽略输入框内的 Z/Y。
- 属性面板使用 `PropertySpec[]` 声明式数组定义每个字段（section/property/label/kind/input/options/unit），渲染器据此动态生成表单控件。
- 变更去重用签名函数（如 `getChangeSignature`、`getConfigMarkSignature`）将对象序列化为 JSON 字符串后比对，避免浅比较遗漏嵌套字段。
- 异步副作用一律用 `RefObject` 缓存最新值并在回调中读取（如 `codeRef`、`schemaRef`、`activeDemoIdRef`），防止闭包过期导致 stale state。
