本模块由三个相互独立的 React 客户端弹窗组件组成，均通过 `Dialog` + `DialogContent` 组合以受控 `open`/`onOpenChange` 形式暴露：
- `ImportFromFigmaDialog.tsx`：负责从剪贴板或 `.html/.htm` 文件读取内容，调用 `parseFigmaImportContent` 解析为 prototype-html-css 或旧版 Markdown 两种格式，再通过 `projectApiClient.createDemoPage` + `updateDemoPageFiles` 写入项目并回调 `onPageCreated`。
- `save-template-dialog.tsx`：表单型弹窗，收集 category/name/description 后通过父级传入的 `onSave` 回调保存模板。
- `KnowledgeDocDialog.tsx`：支持 read/edit/add 三种模式，编辑模式下通过 `useCollabDocument` 与 `/api/knowledge` REST 接口协作；新增走 POST `/api/knowledge`，更新走 PUT `/api/knowledge/:id`，读取走 GET `/api/knowledge/content`；阅读侧使用 `streamdown`（含 code/cjk 插件）渲染 Markdown。
依赖方向单向向外：仅消费 `@/components/ui/*`、`@workbench/shared`、`@workbench/demo-ui` 及上层 hooks/API，不反向依赖 demo 业务逻辑。
