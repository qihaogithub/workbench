本模块是 Next.js App Router 的路由层，仅包含三个扁平的 `page.tsx` 文件：
- `explore/page.tsx` 与 `wish/page.tsx` 作为薄壳，直接导入并渲染 `@/components/explore/explore-page` 与 `@/components/wish/wish-page` 组件。
- `cli/page.tsx` 稍复杂：通过 `next/headers` 读取请求头以获取作者站点 URL（经 `@/lib/project-cli-url.getProjectCliAuthorSiteUrl`），再从 `@workbench/project-core` 生成 CLI 用法提示与快速参考，最终交给 `@/components/cli/project-cli-page` 渲染；同时声明 `export const dynamic = "force-dynamic"` 强制运行时执行。
依赖方向单向向下：路由 → components → lib/core，本模块不持有业务逻辑。
