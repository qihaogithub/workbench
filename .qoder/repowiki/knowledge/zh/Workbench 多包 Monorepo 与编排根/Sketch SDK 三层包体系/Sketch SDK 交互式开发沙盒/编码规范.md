- 所有交互入口组件使用 `'use client'` 指令并以 `export function` 命名导出，页面级组件保持最小职责（page.tsx 仅返回一个顶层组件）。
- 对可能抛错的 JSON 解析统一包裹 try/catch，并通过 `setMessage` 将错误信息写入共享消息区而非抛出。
- 长耗时操作通过 `measureMs(() => {...})` 包装并用 `performance.now()` 计时，结果汇总到 `PerformanceRow` 表格中展示。
- 场景变更通过 `updateScene(previous => next)` 的函数式 setState 回调记录差异摘要，再同步更新 `sceneJson` 文本副本。
- Fixture 数据通过工厂函数 `scene(nodes, pageSize?)` 构造，统一注入 `version: 1`、`assets: []`、`bindings: {}` 等必填字段。
