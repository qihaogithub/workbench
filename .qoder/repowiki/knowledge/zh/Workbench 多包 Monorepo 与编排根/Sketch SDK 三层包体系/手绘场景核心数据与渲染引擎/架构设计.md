模块以单文件 `src/index.ts` 作为唯一入口，按职责自顶向下组织：
- 顶层常量与类型声明（`SKETCH_SCENE_PROTOCOL_VERSION`、`SketchSceneNodeType`、`SketchSceneNode`、`SketchSceneDocument`、`SketchScenePatchOperation` 等）构成对外契约；配套的 `SKETCH_SCENE_*_VALUES` 白名单 Set 用于枚举校验。
- 输入层：`parseSketchSceneDocument` / `normalizeSketchSceneDocument` 负责 JSON→文档的解析与缺省值回填。
- 验证层：`validateSketchSceneDocument` 对版本、pageSize、节点几何/样式/绑定/连接/循环引用等进行全量检查，返回带 code/message/severity 的 `SketchSceneValidationResult`。
- 状态变更层：`applySketchScenePatchOperations` 基于不可变操作集（add/update/delete/duplicate/reorder/group/ungroup/set-locked/set-visible/bind/unbind）批量修改 nodes，并通过 `commitNodesIfValid` 在每次变更后调用验证器做原子性回滚；`applySketchScenePatchOperationsWithResult` 额外产出 `SketchScenePatchSummary` 差异摘要。
- 计算工具层：`getSketchNodeBounds` / `getSketchSelectionBounds` / `computeSketchSnapResult`（含网格吸附）、`hitTestSketchScene`（支持 line/arrow/path/diamond 精确命中）、`translateSketchNodes` / `resizeSketchNode` / `rotateSketchNode`。
- 渲染层：`renderSketchSceneToSvgMarkup` 将文档输出为 SVG 字符串，`buildSketchScenePreviewDocumentHtml` 生成可独立打开的预览 HTML；所有用户可控文本均经 `escapeHtml`/`escapeAttr` 转义。
依赖方向单向：上层 API 仅依赖下层纯函数，无外部运行时依赖，package.json 中 `main`/`types`/`exports` 直接指向 `src/index.ts`，由上层包按需引入。
