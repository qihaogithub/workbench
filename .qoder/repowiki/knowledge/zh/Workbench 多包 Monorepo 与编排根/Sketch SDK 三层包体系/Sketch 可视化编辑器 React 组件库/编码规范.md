- 所有对外接口均以 `interface ...Props` 形式声明，并通过扩展 `SketchEditorPartProps` 复用 scene/controller/className 公共属性。
- 类名合并统一使用本地 `cn(...inputs)` 函数（`twMerge(clsx(inputs))`），避免在 JSX 中直接拼接字符串。
- 场景数据入口遵循 `parseScene` 模式：先尝试解析并校验，失败则回退到 `createDefaultSketchScene()`，保证组件始终处于合法状态。
- 节点可见性与绑定值解析统一通过 `resolveSketchSceneBindingValue(node, field, defaultValue, configData)` 访问，而非直接读取 node 字段。
- 事件处理通过 `data-sketch-node-id` / `data-sketch-node-label` 等 data-* 属性定位目标节点，再用 `closest` 向上查找。
