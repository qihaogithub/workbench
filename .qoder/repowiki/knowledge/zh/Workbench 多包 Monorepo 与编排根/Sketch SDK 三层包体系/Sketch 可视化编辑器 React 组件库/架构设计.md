模块以单文件大组件为主：`src/index.tsx` 是完整的交互式编辑器实现（约 7000+ 行），包含工具状态机（select/hand/rect/diamond/ellipse/line/arrow/pencil/text/image/sticky/eraser）、拖拽/缩放/平移/橡皮擦/框选/快捷键等全部交互逻辑，并通过 `SketchEditorController` 暴露 undo/redo/applyOperations/commitScene 等命令式 API；`src/preview.tsx` 是精简只读预览入口，仅渲染 SVG + 选择叠加层。两者都依赖 `@workbench/sketch-core` 完成场景解析、几何计算与 SVG 序列化，自身不持有持久化状态，通过 props 双向同步 scene 与 selection。包通过 `package.json` 的 `exports` 字段同时导出根路径（完整编辑器）和 `/preview`（轻量预览），peerDependencies 锁定 React ≥18，样式统一使用 clsx + tailwind-merge 的 `cn()` 合并策略，图标来自 lucide-react。测试位于 `tests/sketch-react.test.tsx`，由 vitest 驱动。
