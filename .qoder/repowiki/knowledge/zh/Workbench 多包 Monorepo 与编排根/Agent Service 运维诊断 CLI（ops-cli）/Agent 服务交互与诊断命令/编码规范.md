- 每个命令导出一个异步函数，统一以 `baseUrl` 作为首参，并在内部用 `baseUrl.replace(/\/+$/, '')` 规范化尾部斜杠后再拼接 API 路径。
- 所有用户可见输出使用 chalk 语义色（cyan 标题/灰色元信息/绿色成功/红色错误/黄色提示），并通过 `createSpinner` 包裹耗时操作，在成功/失败分支显式 `spinner.stop()`。
- 当 `jsonMode` 为真时直接 `outputJson(...)` 并提前 return，否则走人类可读的 `showSuccess/showError` 输出，形成“机器可读 / 人类可读”双分支模式。
- 异常处理采用 try/catch 包裹请求块，catch 中同样区分 jsonMode 与非 jsonMode 两种输出路径，并在非 jsonMode 下打印“可能的原因”引导信息后 `process.exit(1)`。
