本模块由三个并列的命令函数组成，均位于 `OPS/CLI/src/commands/` 下，作为上层命令路由的叶子实现：
- `health.ts` 暴露 `healthCheck(baseUrl, jsonMode)`，通过 GET `/health` 拉取后端状态并以 JSON 或彩色文本输出。
- `http-message.ts` 暴露 `testHttpMessage(baseUrl, options, jsonMode)`，复用 `../utils.js` 的 `request` 封装 POST 到 `/api/agent/${sessionId}/message`，返回 `AgentResult`。
- `interactive.ts` 暴露 `interactiveMode(baseUrl, options)`，基于 Node `readline` 构建 REPL；根据 `useWebSocket` 在 HTTP 单轮调用与 `ws` 库的 `/api/agent/${sessionId}/stream` 流式通道之间切换，按 `StreamEvent.type`（stream/plan/finish/error/status）分派渲染。
依赖方向单向指向 `../utils.js`（spinner、showSuccess/showError/outputJson/formatDuration）和 `../types.js`（HttpMessageOptions、InteractiveModeOptions、HealthStatus、StreamEvent、AgentResult），不反向依赖其他命令文件，便于被顶层 CLI 注册为独立子命令。
