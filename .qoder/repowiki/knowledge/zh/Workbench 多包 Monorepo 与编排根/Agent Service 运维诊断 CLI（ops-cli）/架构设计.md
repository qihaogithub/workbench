src/index.ts 作为唯一入口，通过 `new Command()` 声明全局选项 `-u/--url`、`--json`，并以 `program.command(...).action(async () => ...)` 的形式将每个子命令路由到 `src/commands/*.ts` 中的导出函数；各命令文件按职责拆分为 session_commands、diagnostics_health、file_workspace、system_models、http_interactive 五个子模块，彼此不互相依赖，仅由 index.ts 聚合装配。所有命令统一通过传入的 `url` 参数调用 Agent Service 的 REST/WS 接口，并通过 `getJsonMode()` 控制 JSON 输出模式，形成“入口装配 → 命令函数 → 网络调用”的单向分层。
