本模块是纯声明式技能集合，每个子目录是一个独立 skill，由上层 Agent 框架通过 `SKILL.md` 的 YAML front-matter `name`/`description`/`allowed-tools` 字段发现并加载。
- 入口与元数据：每个 skill 根目录的 `SKILL.md` 使用 YAML front-matter 声明 name、description、allowed-tools；部分 skill（creative-page-migrator、workbench-project-admin）额外在 `agents/openai.yaml` 中注入工具映射，供 OpenAI 兼容层解析。
- 参考文档分层：`references/` 存放细粒度参考（如 playwright-cli 下的 element-attributes、tracing、storage-state 等），主 SKILL.md 仅编排工作流并通过相对路径引用，避免单文件膨胀。
- 脚本与运行时：`头脑风暴/scripts/` 包含 Node.js server.cjs、helper.js 及 shell 启动/停止脚本，作为视觉伴生服务的本地运行时，由 SKILL.md 在用户同意时按需拉起。
- 依赖方向：skill 之间通过“推荐调用另一个 skill”的文字约定协作（如 creative-page-migrator 推荐 opencode-project-admin 和 playwright-cli），无代码级耦合；对外依赖 CLI（`ow`、`playwright-cli`、`corepack pnpm`）与文件系统（docs/项目文档、docs/复盘文档）。
