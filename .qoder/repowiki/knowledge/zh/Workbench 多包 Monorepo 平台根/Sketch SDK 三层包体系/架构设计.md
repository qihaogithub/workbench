三个子包通过 pnpm workspace 的 `workspace:*` 协议串联成严格的单向依赖：
- sketch-core 作为零运行时依赖的纯 TS 包，暴露 SketchSceneDocument 协议、节点模型、补丁应用、吸附对齐、命中测试与 SVG 渲染等核心能力。
- sketch-react 声明 peerDependencies react/react-dom，仅依赖 sketch-core，将核心能力封装为 SketchPageEditor、SketchPagePreview 等可组合 React 组件。
- sketch-playground 同时依赖 core 与 react，基于 Next.js 14 + Tailwind 搭建可视化调试与性能基准测试页面，是上层消费方而非被依赖方。
三者均使用 Vitest（playground 用 tsc --noEmit 代替）进行类型检查与单元测试，共享 TypeScript 5.3 与 ESM 模块系统，构成从内核到 UI 再到应用的清晰分层。
