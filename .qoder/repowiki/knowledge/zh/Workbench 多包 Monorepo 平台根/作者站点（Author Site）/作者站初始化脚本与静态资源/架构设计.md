本模块由三类互不依赖的静态/脚本内容组成：
- `scripts/`：基于 ts-node 的一次性 CLI 工具，通过相对路径直接 import `../src/lib/db/schema` 中的 `initializeDatabase` 完成数据库建表；`migrate-sessions.ts` 将旧版 `data/sessions/{projectId}/{sessionId}` 结构扁平迁移为新版 `data/sessions/{userId}/{projectId}/{sessionId}`，并通过向上遍历 `pnpm-workspace.yaml` 定位项目根。
- `public/preview-runtime/`：构建期产出的浏览器端运行时代码，`manifest.json` 以 `imports` 字段声明 `react`、`lucide-react`、`framer-motion`、`@preview/sdk` 等包到 `/preview-runtime/vendor/*.js` 的映射，供预览沙箱按版本加载；`thumbnails/` 存放项目封面图（命名形如 `proj_{timestamp}.png`）。
- `data/ai-attachments/`：AI 对话附件的磁盘存储，按 `session-{id}-{short}/ {attachmentId}/` 两级目录组织，每个附件目录包含 `manifest.json`（记录 id、mimeType、sha256、textExtracted、truncated、originalFilename/storedFilename 等元信息）以及实际文件（如 `____.html`、`text.txt`）。
