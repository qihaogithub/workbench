- UI 组件统一采用 `cva` 定义变体 + `forwardRef` 转发 ref + `displayName` 命名，并通过 `cn(buttonVariants(...), className)` 合并类名。
- 需要副作用或 DOM 访问的组件在文件顶部显式标注 `'use client'` 或 `"use client"`（两种引号风格并存）。
- Context 提供者配套一个同名 Hook（如 `useTheme`、`useToast`），在 `useContext` 返回 undefined 时主动抛错，强制调用方处于 Provider 作用域内。
- 业务组件通过相对路径 `@/components/ui/*` 引用原子组件，避免从 `@/components` 根通配导入，保持 ui 层作为被消费方的稳定契约。
