本模块是 CLI 的命令层，三个文件各自对应一个导出函数：`listSessions`（GET `/api/sessions`，支持 limit/offset/status 分页与过滤）、`getSessionInfo`（GET `/api/agent/:sessionId`）、`destroySession`（DELETE `/api/agent/:sessionId`）。每个命令统一通过 `../utils.js` 中的 `request` 发起 HTTP 调用，使用 `createSpinner` 显示加载状态，并通过 `showSuccess`/`showError` 输出结果；当 `jsonMode=true` 时改走 `outputJson` 输出结构化 JSON，否则以 chalk 彩色表格打印。类型定义复用自 `../types.js` 的 `AgentInfo`。依赖方向单向指向 `utils` 与 `types`，不反向依赖上层路由或入口。
