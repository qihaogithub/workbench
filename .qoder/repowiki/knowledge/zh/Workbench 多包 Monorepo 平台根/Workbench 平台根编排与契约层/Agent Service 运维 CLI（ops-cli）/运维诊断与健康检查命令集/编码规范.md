- 对外入口统一为 async 函数，接收 `(baseUrl|options, options, jsonMode)` 三参，并在 jsonMode 分支直接 `outputJson(result)` 返回结构化结果，非 jsonMode 才走 `console.log` + chalk 彩色输出。
- 所有外部 I/O（HTTP fetch/request、SQLite 打开、SSH 子进程、文件系统读写）都包裹 try/catch，失败时设置 result 对应字段标记 checked=false/exists=false/sent=true+success=false，并输出警告而非抛错中断流程。
- 用户输入参数统一通过 `parseSince`/`toLimit` 等转换函数做归一化与边界限制，避免原始字符串直接进入 SQL 或数组切片。
- 文本输出一律通过 `showSuccess/showError/showWarning/showInfo` 配合 `chalk.gray/yellow/red` 前缀，禁止直接裸写 console.log 作为状态提示。
