# 独立知识库服务

> 更新日期：2026-07-23

## 模块定位

独立知识库服务负责把模板项目内容整理成可跨项目检索、可回读原文、可追溯到项目修订的知识资产。Agent 先读取当前项目知识，再按需查询其他模板项目；其他模板只能作为参考，不能覆盖当前项目事实。

创作端的知识库面板仍保留在 [创作端/09-知识库](../创作端/09-知识库/)；本目录只描述跨端复用的知识库服务边界。
