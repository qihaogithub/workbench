# 独立知识库服务

> 更新日期：2026-06-26

## 模块定位

独立知识库服务负责把系统规则、当前项目知识、模板知识、模板库参考和会话上下文整理成 AI 可检索、可解释、可追溯、可按权限过滤的知识资产。

创作端的知识库面板仍保留在 [创作端/09-知识库](../创作端/09-知识库/)；本目录只描述跨端复用的知识库服务边界。
