## 核心概念

### 嵌入模式

外部系统通过 iframe 将创作端的 Demo 组件嵌入到自身页面中，实现组件的共享与分发。

### 双向通信

通过 `postMessage` API 实现父页面与 iframe 的双向通信，支持代码更新、配置更新、错误上报等功能。
