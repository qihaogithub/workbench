# 预览系统 - 需求文档

> 版本：v2.0
> 创建日期：2026-04-06
> 更新日期：2026-04-19

---

## 一、模块定位

预览系统是 Demo 可视化呈现的核心模块，负责在浏览器端实时渲染 AI 生成的 React 组件，提供所见即所得的预览体验。本模块采用**服务端编译 + 前端直接渲染**架构，确保用户配置的变更能够即时反映在预览区，缩短反馈循环。

---

## 二、功能描述

### 2.1 实时预览功能

#### 2.1.1 组件渲染

**用户视角**：
- 系统应在预览区实时渲染 AI 生成的 React 组件
- 渲染结果应与实际使用效果一致

**交互效果**：
- 预览区应支持 React 组件的完整渲染
- 预览区应支持 TypeScript/JSX 代码的编译执行
- 组件内部的交互行为（如点击、动画）应正常工作
- 预览组件应直接使用项目的样式系统和组件库（shadcn/ui）

#### 2.1.2 配置注入

**用户视角**：
- 用户在配置面板设置的参数应自动注入到 Demo 组件
- 配置变更后，预览应即时更新

**交互效果**：
- 配置值应作为组件 Props 传递
- 配置变更后，预览应在 < 16ms 内更新（一帧内响应）

#### 2.1.3 错误展示

**用户视角**：
- 当代码存在语法错误时，系统应在预览区显示错误信息
- 当组件运行时发生错误时，系统应显示错误边界提示

**交互效果**：
- 错误信息应包含错误类型和位置
- 错误信息应使用用户友好的语言描述
- 应提供错误修复建议（如可能）

---

### 2.2 预览尺寸控制功能

#### 2.2.1 自定义尺寸

**用户视角**：
- 用户可设置预览区的宽度和高度
- 预览区应支持响应式布局

**交互效果**：
- 支持设置固定尺寸（如 800x600 像素）
- 支持设置百分比尺寸（如 100%宽度）
- 支持设置最小/最大尺寸限制

#### 2.2.2 尺寸预设

**用户视角**：
- 系统应提供常用设备尺寸预设
- 用户可快速切换不同设备预览

**交互效果**：
- 预设尺寸应包括：手机、平板、桌面等常见设备
- 切换预设后，预览区应立即调整尺寸

---

### 2.3 预览刷新功能

#### 2.3.1 手动刷新

**用户视角**：
- 用户可手动触发预览刷新
- 刷新后，预览区应重新编译并渲染组件

**交互效果**：
- 提供刷新按钮
- 编译过程中显示加载状态

#### 2.3.2 自动刷新

**用户视角**：
- 代码变更后，预览应自动重新编译并刷新
- 配置变更不触发重新编译，直接通过 props 更新
- 自动刷新应支持缓存，相同代码不重复编译

**交互效果**：
- 代码变更后自动触发编译和刷新
- 编译结果应缓存，提升重复加载速度

## 三、业务规则与约束

### 3.1 执行环境约束

| 约束 | 说明 |
|:-----|:-----|
| 依赖白名单 | AI 生成的组件只能使用项目已有依赖（react、lucide-react、clsx 等） |
| 代码来源 | 组件代码由 AI 生成，经过系统提示词约束和白名单校验 |
| 存储限制 | 预览环境无法访问本地存储 |

### 3.2 支持的技术栈

| 技术 | 版本要求 | 说明 |
|:-----|:---------|:-----|
| React | 18.x | 核心渲染框架 |
| TypeScript | 5.x | 类型安全的代码编写 |
| CSS | - | 支持 Tailwind CSS、CSS Modules |
| 组件库 | shadcn/ui | 项目内置组件库，可直接使用 |

### 3.3 性能要求

| 指标 | 要求 | 说明 |
|:-----|:-----|:-----|
| 首次渲染时间 | < 1s | 从加载到首次渲染完成（含编译） |
| 配置联动时间 | < 16ms | 配置变更到预览更新（无需重新编译） |
| 代码编译时间 | < 500ms | 代码变更到编译完成 |
| 错误恢复时间 | < 1s | 错误修复到预览恢复 |

## 四、预览尺寸配置规范

### 4.1 尺寸配置格式

在配置定义中通过扩展字段设置预览尺寸：

```json
{
  "$demo": {
    "previewSize": {
      "width": "100%",
      "height": "600px",
      "minHeight": "400px",
      "maxHeight": "800px"
    }
  }
}
```

### 4.2 预设尺寸列表

| 预设名称 | 宽度 | 高度 | 适用场景 |
|:---------|:-----|:-----|:---------|
| 手机竖屏 | 375px | 667px | 移动端应用 |
| 手机横屏 | 667px | 375px | 移动端游戏 |
| 平板竖屏 | 768px | 1024px | 平板应用 |
| 平板横屏 | 1024px | 768px | 平板应用 |
| 桌面 | 100% | 100% | 桌面应用 |
| 自定义 | 用户定义 | 用户定义 | 特殊需求 |
