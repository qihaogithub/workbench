## 历史文档（保留参考）

本文档保留作为历史参考，描述 v1.0 架构中使用端如何通过 Schema 解析和动态表单生成实现配置面板。

### 原始内容摘要

- **Schema 解析**：`parseSchemaToFields()` 将 JSON Schema 转换为字段列表
- **控件映射**：string/number/boolean/array 映射到不同 Radix UI 控件
- **变更流程**：使用者修改 → `setFormData()` → `onChange()` → `postMessage(UPDATE_CONFIG)`
- **重置功能**：`extractDefaults(schema)` 获取默认值并重置

> 完整历史内容已归档，新开发请使用创作端 viewer 内置的配置面板。
