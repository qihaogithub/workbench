# 创作端画布自由标注工具

## 背景

创作端画布工作台已支持页面总览、文档节点、图片节点，以及本地 Markdown/图片拖拽导入。当前需要补齐工具栏入口，让用户可直接添加文档、文字、箭头、画笔和图片。

## 目标

- 在画布工具栏提供文档、文字、箭头、画笔、图片入口。
- 复用已有图片本地读取能力，通过按钮选择本地图片。
- 将文字、箭头、画笔作为画布自由标注节点持久化。

## 范围

- 修改共享画布类型、画布组件、自由节点渲染、画布布局 API 校验和相关测试。
- 更新配置与预览模块的需求文档和技术文档。
- 不将新增自由标注写回 Demo 页面代码，不接入 AI 批注发送链路。

## 方案

- 扩展 `CanvasFreeNodeKind`，新增 `text`、`arrow`、`drawing`。
- 工具栏新增文字、箭头、画笔、图片按钮，保持现有文档按钮。
- 文字和箭头按钮在当前视口中心创建节点；画笔按钮进入绘制模式，在画布空白拖拽后生成折线节点；图片按钮触发隐藏文件选择并复用图片节点创建逻辑。
- API 保存前解析新增节点类型，保持非法布局、非法字段拒绝保存。
