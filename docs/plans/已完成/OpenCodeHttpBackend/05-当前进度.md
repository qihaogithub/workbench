### 3.3 完善测试

**新增测试文件**：`tests/unit/default-backend.test.ts`（9 个测试）

- AgentFactory 默认后端验证
- MemorySessionStore 默认后端验证
- 双后端共存验证
- 按 backend 过滤会话

**增强测试文件**：`tests/unit/opencode-http.test.ts`（33 个测试）

新增测试用例：
- `cancelPrompt` — 取消进行中的流式请求
- `setModel` — 运行时切换模型
- `getModelInfo` — 从服务器获取模型列表（含失败回退）
- SSE `agent_thought_chunk` 事件
- SSE `error` 事件（Promise reject）
- SSE 连接错误（onerror 处理）
- SSE `tool_call_update` 事件
- `start` with resume（恢复已有会话）
- 健康检查失败场景
- `destroy` with active stream — 销毁时拒绝活跃流式请求
- `setPromptTimeout` — 设置超时（含无 opencode config 场景）
- `initialize` idempotency — 防重复初始化
- `sendMessage` sync error handling — 同步/异步消息失败处理
- `sendMessage` without session — 无会话时自动初始化
- `permission handling fallback` — 无 allow 选项时的回退处理
- SSE stream timeout — 流式超时处理
- `getWorkingDir` — 工作目录获取

**关键实现修复**：
- `sendMessageStream` 轮询循环 → `streamDone` Promise 回调（消除竞态条件）
- `handleSSEEvent` 中 `if (!this.eventCallback) return` → `this.eventCallback?.()`（确保无 callback 时 streamDone 仍能 resolve）
- `onerror` 处理器添加 `this.status = 'error'` + `streamDone.reject`
