# 已完成计划文档索引

本目录只保留已经完成、且仍有复用价值的技术方案、问题复盘和任务归档。已完成文档不是过程日志库：不保留逐次尝试、完整命令输出或已失效待办，只保留最终结论、关键证据和可追溯索引。

最后整理时间：2026-07-08

## 维护标准

- 已完成文档应聚焦“结论、影响、验证、剩余风险”，避免继续保留长篇进度流水。
- 已完成且没有复用价值的记录应删除，不为了保留历史而保留历史。
- 如果结论已经进入 `docs/项目文档/`，本目录只保留归档说明和链接，不重复维护事实正文。
- 仍需持续跟踪的问题应放回 `docs/plans/进行中/`，不要长期停留在本目录。
- 新增归档前先检查是否能并入既有主题文档，避免重新膨胀为“一问题一文档”。
