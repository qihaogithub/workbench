# 三栏布局左右栏等宽方案

## 一、需求概述

### 目标
修改 demo 编辑页的三栏布局，调整左右两侧栏的默认宽度为 25%，同时**保留拖拽调整功能**。

### 当前状态

**布局文件**：`packages/web/src/app/demo/[id]/edit/page.tsx`

```tsx
<ResizablePanelGroup
  direction="horizontal"
  defaultSizes={[35, 35, 30]}  // 当前：左35%，中35%，右30%
  minSizes={[20, 20, 20]}       // 最小20%
  className="h-full"
>
```

---

## 二、方案设计

### 方案：调整默认宽度比例（推荐）

**核心思路**：仅修改 `defaultSizes` 参数，将左右两侧改为 25%，中间改为 50%。

**优点**：
- 改动最小，无需修改组件逻辑
- 保留完整的拖拽调整功能
- 左右栏初始等宽（各 25%）

**参数修改**：

| 参数 | 当前值 | 修改值 |
|------|--------|--------|
| `defaultSizes` | `[35, 35, 30]` | `[25, 50, 25]` |
| `minSizes` | `[20, 20, 20]` | `[15, 30, 15]` |

**效果**：
- 初始状态：左侧 25%，中间 50%，右侧 25%
- 用户仍可通过拖拽调整各栏宽度
- 左右栏始终保持初始等宽比例

---

## 三、涉及文件

| 文件 | 操作 |
|------|------|
| `packages/web/src/app/demo/[id]/edit/page.tsx` | 修改 `defaultSizes` 和 `minSizes` |

## 五、验收标准

1. 左侧栏和右侧栏默认宽度为 25%
2. 中间预览区默认宽度为 50%
3. 各栏仍可拖拽调整宽度
4. 中间预览区最小宽度 30%，两侧栏最小宽度 15%
5. 窗口缩放时布局正常
