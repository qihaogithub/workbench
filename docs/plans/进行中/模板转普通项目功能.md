# 模板转普通项目功能

## 背景

首页模板卡片当前支持使用模板新建、修改名称、修改分类、修改封面和删除。用户希望在模板项目菜单中增加“转为普通项目”，把一个模板条目恢复为普通项目。

## 目标

- 在模板卡片更多菜单中增加“转为普通项目”入口。
- 转换时按模板快照内容创建一个普通项目，并移除原模板条目。
- 转换后刷新首页普通项目列表与模板列表，并给出成功或失败提示。
- 补充必要测试与项目模板文档。

## 范围

- `packages/project-core/`：提供模板转换为普通项目的领域能力。
- `packages/author-site/`：新增 API 调用、菜单入口和页面交互。
- `docs/项目文档/创作端/03-项目管理/`：同步项目模板需求与技术说明。

## 方案

服务端新增模板转换接口，内部读取模板元数据后调用现有模板实例化流程创建普通项目，再删除模板快照。前端模板卡片菜单触发该接口，成功后刷新 `/api/demos` 与 `/api/templates` 对应缓存。
