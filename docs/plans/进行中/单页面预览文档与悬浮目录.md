# 单页面预览文档与下拉目录

## 背景

创作端项目编辑页的画布模式已经可以查看页面和文档，但单页面预览模式只能查看当前 Demo 页面。目录入口最终保留在单页模式右上角下拉中，并扩展为页面与文档统一选择入口。

## 目标

- 单页面模式支持在页面和文档之间切换查看。
- 单页右上角页面下拉扩展为页面和文档统一目录。
- 文档预览复用画布文档节点的 Markdown 浏览能力。
- 当前选中为文档时隐藏右侧配置与属性面板。
- 同步更新配置与预览模块文档。

## 范围

- `packages/demo-ui/` 文档节点浏览组件与内容读取逻辑。
- `packages/author-site/src/app/demo/[id]/edit/page.tsx` 单页目标状态、目录和右侧面板显示规则。
- `packages/author-site/src/components/demo/` 相关测试。
- `docs/项目文档/创作端/04-配置与预览/` 对应需求与技术文档。

不变更 `.canvas-layout.json` 结构、知识文档 API、页面 Schema 或画布持久化协议。

## 方案

- 将 `CanvasFreeNodeItem` 中的文档 Markdown 渲染和聚合文档目录切换抽成可复用组件。
- 将 `PreviewCanvas` 中的知识文档 Markdown 读取缓存抽成共享 hook。
- 编辑页新增单页预览目标状态：页面目标继续驱动 `PreviewPanel`，文档目标只在单页预览区渲染文档内容。
- 右上角下拉目录按页面和文档分组，页面条目走现有页面加载流程，文档条目只切换单页文档目标。
- 文档条目来自画布文档节点和未隐藏知识文档，避免聚合文档与已隐藏单文档重复展示。
