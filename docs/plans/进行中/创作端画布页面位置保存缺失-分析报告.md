# 创作端画布页面位置保存缺失-分析报告

## 背景

用户反馈：在创作端画布中修改页面位置后，保存按钮没有亮起，页面在画布中的位置信息似乎不会保存。

## 目标

- 定位页面画布位置变更没有触发保存状态的根因。
- 确认保存 payload 是否包含页面位置信息。
- 在最小范围内修复，并补充匹配验证。

## 范围

- 创作端编辑页画布交互、dirty 状态、保存逻辑。
- shared 画布页面节点类型和事件回调。
- 不处理与本问题无关的项目管理、发布、Agent 或使用端功能改动。

## 方案

1. 收集环境与代码路径证据。
2. 追踪页面位置变更从画布组件到编辑页状态的调用链。
3. 修复位置变更未纳入保存状态或保存数据的问题。
4. 运行与创作端相关的最小验证。
