## 后端选择策略

| 场景 | 推荐后端 | 理由 |
|:---|:---|:---|
| **生产环境（多用户）** | `opencode-http` | 无状态、易部署、系统级配置 |
| **本地开发** | `opencode` (ACP) | 轻量、无需启动 Server |
| **IDE 插件集成** | `opencode` (ACP) | 兼容 ACP 生态（Zed 等） |
| **调试/测试** | `opencode` (ACP) | 进程内调试更方便 |
| **高可用部署** | `opencode-http` | 支持负载均衡、健康检查 |
