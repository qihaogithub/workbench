# 预览区画布模式可行性分析与实施方案

> 版本：v5.0 | 更新日期：2026-06-01 | 方案：服务端 Puppeteer 快照浏览模式（100% 还原度）

## 一、概述

### 需求背景

当前预览区已有两种模式：

- **单页面模式**：单个 iframe 实时预览
- **宫格模式**：多页面等比缩略展示

### 新增需求

新增 **画布模式(Canvas Mode)**：

- **创作端**：无限画布 + 自由拖拽平移/缩放 + **可调整页面位置和尺寸** + 布局自动保存
- **使用端**：无限画布 + 自由拖拽平移/缩放 + **只读布局**（不可调整页面）

### 技术方案：服务端 Puppeteer 快照浏览模式

画布上展示页面快照（PNG 图片），**使用服务端 Puppeteer 真实 Chrome 渲染生成 100% 还原的快照**，双击切换至实时编辑模式。

**核心优势**：

- ✅ **100% 像素级还原**：真实 Chrome 渲染，支持所有 CSS 特性
- ✅ **无需用户授权**：服务端静默批量生成
- ✅ **质量可控**：可设置截图质量、分辨率、等待条件
- ✅ **使用端友好**：客户端仅加载图片，性能极佳

```typescript
// packages/agent-service/src/routes/snapshots.ts

import { Router } from "express";
import puppeteer from "puppeteer";
import { compileCode, generateIframeHtml } from "../utils/compiler";
import { uploadToCDN, getSnapshotCache } from "../utils/storage";

const router = Router();

// Puppeteer 浏览器实例池
let browserPool: puppeteer.Browser[] = [];
const MAX_POOL_SIZE = 3;

async function getBrowser(): Promise<puppeteer.Browser> {
  if (browserPool.length === 0) {
    const browser = await puppeteer.launch({
      headless: "new",
      args: [
        "--no-sandbox",
        "--disable-setuid-sandbox",
        "--disable-dev-shm-usage",
        "--disable-gpu",
      ],
    });
    browserPool.push(browser);
  }
  return browserPool[0];
}

/**
 * 生成页面快照（100% 还原）
 */
router.post("/generate", async (req, res) => {
  const { pageId, code, configData, width = 1200, height = 800 } = req.body;

  try {
    // 1. 检查缓存（代码未变则直接返回）
    const cached = await getSnapshotCache(pageId, code);
    if (cached) {
      return res.json({ success: true, data: { url: cached } });
    }

    // 2. 编译页面代码
    const compiled = await compileCode(code);

    // 3. 生成完整 HTML
    const html = generateIframeHtml({
      compiledCode: compiled.compiledCode,
      cssImports: compiled.cssImports,
      configData: configData || {},
    });

    // 4. Puppeteer 渲染（真实 Chrome）
    const browser = await getBrowser();
    const page = await browser.newPage();

    await page.setViewport({ width, height });

    // 5. 加载 HTML
    await page.setContent(html, {
      waitUntil: "networkidle0", // 等待所有网络请求完成
      timeout: 15000,
    });

    // 6. 等待 Web 字体加载完成
    await page.evaluateHandle("document.fonts.ready");

    // 7. 截图（100% 像素级还原）
    const screenshot = await page.screenshot({
      type: "png",
      quality: 100,
      fullPage: false,
      clip: { x: 0, y: 0, width, height },
    });

    // 8. 关闭页面（保留浏览器实例）
    await page.close();

    // 9. 上传至 CDN
    const snapshotUrl = await uploadToCDN(
      screenshot,
      `snapshots/${pageId}.png`,
    );

    // 10. 更新缓存
    await setSnapshotCache(pageId, code, snapshotUrl);

    res.json({ success: true, data: { url: snapshotUrl } });
  } catch (error) {
    console.error("生成快照失败:", error);
    res.status(500).json({
      success: false,
      error: { code: "SNAPSHOT_ERROR", message: "生成快照失败" },
    });
  }
});

/**
 * 批量生成快照（并发控制）
 */
router.post("/generate-batch", async (req, res) => {
  const { pages } = req.body; // Array<{ pageId, code, configData }>

  try {
    const concurrency = 5; // 同时渲染 5 个页面
    const results: Record<string, string> = {};

    // 并发处理
    for (let i = 0; i < pages.length; i += concurrency) {
      const batch = pages.slice(i, i + concurrency);
      const promises = batch.map(async ({ pageId, code, configData }) => {
        try {
          const response = await generateSnapshot(pageId, code, configData);
          results[pageId] = response.url;
        } catch (error) {
          console.error(`快照生成失败 [${pageId}]:`, error);
          results[pageId] = generatePlaceholderUrl(pageId);
        }
      });

      await Promise.all(promises);
    }

    res.json({ success: true, data: { urls: results } });
  } catch (error) {
    res.status(500).json({
      success: false,
      error: { code: "BATCH_ERROR", message: "批量生成失败" },
    });
  }
});

export default router;
```

#### 3.1.2 快照缓存策略

```typescript
// packages/agent-service/src/utils/storage.ts

import { createHash } from "crypto";

interface SnapshotCacheEntry {
  url: string;
  generatedAt: number;
  codeHash: string;
  expiresAt: number;
}

const SNAPSHOT_TTL = 24 * 60 * 60 * 1000; // 24 小时
const snapshotCache = new Map<string, SnapshotCacheEntry>();

/**
 * 检查快照缓存是否有效
 */
export async function getSnapshotCache(
  pageId: string,
  code: string,
): Promise<string | null> {
  const cached = snapshotCache.get(pageId);
  if (!cached) return null;

  if (Date.now() > cached.expiresAt) {
    snapshotCache.delete(pageId);
    return null;
  }

  const currentHash = createHash("md5").update(code).digest("hex");
  if (currentHash !== cached.codeHash) {
    snapshotCache.delete(pageId);
    return null;
  }

  return cached.url;
}

/**
 * 设置快照缓存
 */
export async function setSnapshotCache(
  pageId: string,
  code: string,
  url: string,
): Promise<void> {
  const codeHash = createHash("md5").update(code).digest("hex");

  snapshotCache.set(pageId, {
    url,
    generatedAt: Date.now(),
    codeHash,
    expiresAt: Date.now() + SNAPSHOT_TTL,
  });
}
```

```typescript
// packages/author-site/src/components/preview/PreviewCanvas.tsx

interface PreviewCanvasProps {
  editable: boolean;
  pages: Array<{ id: string; code: string; configData?: any }>;
  canvasState: CanvasState;
  onCanvasStateChange: (state: CanvasState) => void;
}

export function PreviewCanvas({
  editable,
  pages,
  canvasState,
  onCanvasStateChange,
}: PreviewCanvasProps) {
  const [snapshots, setSnapshots] = useState<Record<string, string>>({});
  const [loading, setLoading] = useState(true);

  // 生成快照（调用服务端 API）
  useEffect(() => {
    generateSnapshots();
  }, [pages]);

  const generateSnapshots = async () => {
    setLoading(true);
    try {
      const response = await fetch('/api/snapshots/generate-batch', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          pages: pages.map(p => ({
            pageId: p.id,
            code: p.code,
            configData: p.configData,
          })),
        }),
      });

      const result = await response.json();
      if (result.success) {
        setSnapshots(result.data.urls);
      }
    } catch (error) {
      console.error('生成快照失败:', error);
    } finally {
      setLoading(false);
    }
  };

  return (
    <div className="w-full h-full overflow-hidden">
      {/* 工具栏 */}
      {editable && <CanvasToolbar />}

      {/* 画布视口 */}
      <CanvasViewport
        viewport={canvasState.viewport}
        onViewportChange={(viewport) =>
          onCanvasStateChange({ ...canvasState, viewport })
        }
      >
        {/* 页面快照列表 */}
        {pages.map((page) => (
          <CanvasPageItem
            key={page.id}
            pageId={page.id}
            snapshotUrl={snapshots[page.id]}
            layout={canvasState.pages[page.id]}
            editable={editable}
            onLayoutChange={(layout) =>
              onCanvasStateChange({
                ...canvasState,
                pages: { ...canvasState.pages, [page.id]: layout },
              })
            }
          />
        ))}
      </CanvasViewport>

      {/* 加载状态 */}
      {loading && <SnapshotLoadingIndicator />}
    </div>
  );
}
```

#### 视口容器（缩放/平移）

```typescript
// packages/author-site/src/components/preview/CanvasViewport.tsx

interface CanvasViewportProps {
  viewport: CanvasViewport;
  onViewportChange: (viewport: CanvasViewport) => void;
  children: React.ReactNode;
}

export function CanvasViewport({
  viewport,
  onViewportChange,
  children,
}: CanvasViewportProps) {
  const [isPanning, setIsPanning] = useState(false);
  const [startPos, setStartPos] = useState({ x: 0, y: 0 });

  // 鼠标按下：开始平移
  const handleMouseDown = (e: React.MouseEvent) => {
    if (e.button !== 0) return; // 仅左键
    setIsPanning(true);
    setStartPos({ x: e.clientX - viewport.x, y: e.clientY - viewport.y });
  };

  // 鼠标移动：更新视口位置
  const handleMouseMove = (e: React.MouseEvent) => {
    if (!isPanning) return;
    onViewportChange({
      ...viewport,
      x: e.clientX - startPos.x,
      y: e.clientY - startPos.y,
    });
  };

  // 鼠标释放：结束平移
  const handleMouseUp = () => {
    setIsPanning(false);
  };

  // 滚轮缩放
  const handleWheel = (e: React.WheelEvent) => {
    e.preventDefault();
    const delta = e.deltaY > 0 ? 0.9 : 1.1;
    const newZoom = Math.min(Math.max(viewport.zoom * delta, 0.1), 3);
    onViewportChange({ ...viewport, zoom: newZoom });
  };

  return (
    <div
      className="w-full h-full cursor-grab active:cursor-grabbing"
      onMouseDown={handleMouseDown}
      onMouseMove={handleMouseMove}
      onMouseUp={handleMouseUp}
      onMouseLeave={handleMouseUp}
      onWheel={handleWheel}
    >
      <div
        style={{
          transform: `translate(${viewport.x}px, ${viewport.y}px) scale(${viewport.zoom})`,
          transformOrigin: '0 0',
        }}
      >
        {children}
      </div>
    </div>
  );
}
```

#### 页面快照项（拖拽 + 调整尺寸）

```typescript
// packages/author-site/src/components/preview/CanvasPageItem.tsx

interface CanvasPageItemProps {
  pageId: string;
  snapshotUrl: string;
  layout: CanvasPageLayout;
  editable: boolean;
  onLayoutChange: (layout: CanvasPageLayout) => void;
}

export function CanvasPageItem({
  pageId,
  snapshotUrl,
  layout,
  editable,
  onLayoutChange,
}: CanvasPageItemProps) {
  const [isDragging, setIsDragging] = useState(false);
  const [startPos, setStartPos] = useState({ x: 0, y: 0 });

  // 拖拽移动页面
  const handleDragStart = (e: React.MouseEvent) => {
    if (!editable) return;
    e.stopPropagation(); // 阻止画布平移
    setIsDragging(true);
    setStartPos({
      x: e.clientX - layout.x,
      y: e.clientY - layout.y,
    });
  };

  const handleDrag = (e: React.MouseEvent) => {
    if (!isDragging) return;
    onLayoutChange({
      ...layout,
      x: e.clientX - startPos.x,
      y: e.clientY - startPos.y,
    });
  };

  const handleDragEnd = () => {
    setIsDragging(false);
  };

  // 双击切换至单页模式
  const handleDoubleClick = () => {
    window.location.href = `/preview/${pageId}?mode=single`;
  };

  return (
    <div
      className={`absolute ${editable ? 'cursor-move' : 'cursor-pointer'}`}
      style={{
        left: layout.x,
        top: layout.y,
        width: layout.width,
        height: layout.height,
        zIndex: layout.zIndex,
      }}
      onMouseDown={handleDragStart}
      onMouseMove={handleDrag}
      onMouseUp={handleDragEnd}
      onDoubleClick={handleDoubleClick}
    >
      {/* 快照图片 */}
      <img
        src={snapshotUrl}
        alt={`Page ${pageId}`}
        className="w-full h-full object-cover rounded-lg shadow-lg"
      />

      {/* 创作端：调整尺寸手柄 */}
      {editable && <CanvasResizeHandle onResize={onLayoutChange} />}

      {/* 创作端：选中状态边框 */}
      {editable && isDragging && (
        <div className="absolute inset-0 border-2 border-blue-500 rounded-lg" />
      )}
    </div>
  );
}
```

### 4.5 手势冲突处理

**事件优先级**：

```
1. 尺寸调整手柄（最高）
   └─ e.stopPropagation()

2. 页面拖拽移动（中）
   └─ e.stopPropagation()

3. 画布平移（最低）
```

**实现策略**：

```typescript

  e.stopPropagation(); // 阻止事件冒泡到画布
  // ...
};

// CanvasResizeHandle.tsx

  e.stopPropagation(); // 阻止事件冒泡到页面和画布
  // ...
};
```

## 五、风险与应对

### 5.1 性能风险

| 风险                  | 应对策略                     |
| --------------------- | ---------------------------- |
| 快照生成慢（1-2s/页） | 异步生成 + 骨架屏 + 缓存     |
| CDN 下载慢            | 快照压缩 + CDN 加速 + 懒加载 |
| 内存占用高            | 快照 24 小时过期 + 自动清理  |

### 5.2 兼容性风险

| 风险               | 应对策略                      |
| ------------------ | ----------------------------- |
| Puppeteer 部署问题 | Docker 已提供 Chrome 环境     |
| CSS 渲染差异       | 100% 真实 Chrome 渲染，无差异 |
| Web 字体加载       | 等待 `document.fonts.ready`   |

### 5.3 用户体验风险

| 风险             | 应对策略                         |
| ---------------- | -------------------------------- |
| 快照不是实时内容 | 明确 UI 提示"双击查看实时内容"   |
| 快照过期         | 自动重新生成 + 缓存策略          |
| 手势冲突         | Pointer Capture API + 事件优先级 |

## 七、总结

### 技术选型对比

| 方案                 | 还原度   | 性能           | 复杂度 | 推荐度     |
| -------------------- | -------- | -------------- | ------ | ---------- |
| **服务端 Puppeteer** | **100%** | 中（1-2s/页）  | 中     | ⭐⭐⭐⭐⭐ |
| 客户端 dom-to-image  | 95-98%   | 快（200ms/页） | 低     | ⭐⭐⭐     |
| 客户端 html2canvas   | 90-95%   | 快（200ms/页） | 低     | ⭐⭐       |

### 最终决策

**选择服务端 Puppeteer 方案**，核心原因：

1. ✅ **100% 还原度**（不可妥协）
2. ✅ 支持所有 CSS 特性
3. ✅ 无需用户授权
4. ✅ 使用端友好（仅加载图片）
5. ⚠️ 快照生成耗时可接受（异步 + 缓存）
