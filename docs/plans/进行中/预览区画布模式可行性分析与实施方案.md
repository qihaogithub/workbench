# 预览区画布模式可行性分析与实施方案

> 版本：v6.0 | 更新日期：2026-06-03 | 方案：图片（Puppeteer 截图）+ 单 Iframe 混合画布

## 一、概述

### 需求背景

当前预览区已有两种模式：

- **单页面模式**：单个 iframe 实时预览
- **宫格模式**：多页面等比缩略展示

### 新增需求

新增 **画布模式(Canvas Mode)**：

- **创作端**：无限画布 + 自由拖拽平移/缩放 + **可调整页面位置** + 布局自动保存
- **使用端**：无限画布 + 自由拖拽平移/缩放 + **只读布局**（不可调整页面）

### 技术方案：图片 + 单 Iframe 混合画布

**核心思路**：画布上默认展示 Puppeteer 生成的页面截图（`<img>`），当用户需要修改某页配置时，仅该页切换为 `<iframe>` 实时预览，配置完成后切回图片。

```
画布状态：50 张 <img> 截图 → 流畅 60fps
     ↓ 鼠标悬浮某页
出现「修改配置」按钮（创作端和使用端均显示）
     ↓ 点击
该页 <img> → 切换为 <iframe>（实时预览）
配置面板同步打开，编辑时即时看到效果
其他 49 页保持 <img> 不变
     ↓ 关闭配置 / 点击画布空白
<iframe> 移除 → 恢复原来 <img>（不重新截图）
画布恢复纯图片模式
```

### 为什么这个方案优雅

| 场景 | 实际挂载 | 性能 |
|------|---------|------|
| 全局缩略（50页全部可见） | 0 个 iframe，50 张 `<img>` | ✅ 60fps |
| 放大查看局部 | 0 个 iframe，N 张 `<img>` | ✅ 60fps |
| **编辑某一页配置时** | **恰好 1 个 iframe** | ✅ 该页实时 + 其余不变 |

**Puppeteer 只负责离线截图，浏览器只跑一个 iframe，互不干扰。**

### 核心优势

- ✅ **100% 像素级还原**：Puppeteer 真实 Chrome 渲染截图
- ✅ **同时可见 50 页**：`<img>` 标签无性能压力
- ✅ **编辑时有实时预览**：单 iframe 实时响应配置变更
- ✅ **无需用户授权**：服务端静默批量生成截图
- ✅ **布局持久化**：位置/尺寸自动保存到服务端

## 二、核心方案设计

### 2.1 双端权限模型

| 功能 | 创作端 | 使用端 |
|------|--------|--------|
| 拖拽平移画布 | ✅ | ✅ |
| 缩放画布 | ✅ | ✅ |
| 拖拽调整页面位置 | ✅ | ❌ |
| 调整页面尺寸 | ✅ | ❌ |
| 悬浮查看修改配置按钮 | ✅ | ✅ |
| 点击修改配置（切换 iframe） | ✅ | ✅ |
| 返回画布模式 | ✅ | ✅ |

通过单一 `editable` prop 控制：

```typescript
// 创作端
<PreviewCanvas editable={true} ... />

// 使用端
<PreviewCanvas editable={false} ... />
```

### 2.2 截图架构

```
编辑页代码保存/变更
   ↓
后台自动触发 POST /api/snapshots/generate (单页截图)
Agent-service Puppeteer 渲染截图 → 保存至 data/snapshots/{projectId}/{pageId}.png
   ↓
用户打开画布模式
   ↓
GET /api/snapshots/list/{projectId} → 列出已有截图 URL
   ↓
Client 直接用 <img> 标签渲染
   ↓
画布模式即开即用（所有页面截图已就绪）
```

**截图生成时机**：
- 页面代码变更时 → 单页截图（后台异步，不阻塞编辑）
- 打开画布模式时 → 仅加载已有截图，不触发新生成
- 若截图不存在 → 显示加载占位，触发后台生成

**配置编辑与截图的关系**：
- 点击修改配置 → iframe 实时预览（不重新截图）
- 关闭 iframe → 恢复原来的截图（不重新截图）
- 只有代码变更才会触发新截图

### 2.3 配置编辑流程（iframe 切换）

```
用户悬浮 Page-A 的 <img> 块
   ↓
「修改配置」按钮浮现
   ↓ 点击
1. Page-A 的 <img> 隐藏，同级渲染 <PreviewPanel>（iframe）
2. 配置面板切换到 Page-A 的 Schema
3. 用户在配置面板修改 → PostMessage → iframe 实时更新
4. 其他 49 页仍是 <img>
   ↓ 点击画布空白 / 关闭配置面板
1. <iframe> 移除 → <img> 恢复（原始截图，不重新生成）
2. 画布恢复纯图片模式
```

**重要**：配置编辑期间的修改 **不会** 触发截图重新生成。截图仅在页面代码变更时生成。配置编辑器仅用于在画布上临时预览效果。如需保存配置效果，用户需在页面编辑模式下进行。

#### 4.3.1 截图生成 API（agent-service）

```typescript
// packages/agent-service/src/routes/snapshots.ts

import { FastifyInstance, FastifyRequest, FastifyReply } from "fastify";
import { readFileSync, existsSync, mkdirSync, writeFileSync } from "fs";
import { join } from "path";
import { getSnapshotRenderer } from "../utils/snapshot-renderer";

interface GenerateSnapshotBody {
  projectId: string;
  pageId: string;
  code: string;
  schema?: string;
  configData?: Record<string, unknown>;
  width?: number;
  height?: number;
}

interface GenerateBatchBody {
  projectId: string;
  pages: Array<{
    pageId: string;
    code: string;
    schema?: string;
    configData?: Record<string, unknown>;
    width?: number;
    height?: number;
  }>;
}

export async function registerSnapshotRoutes(fastify: FastifyInstance) {
  const dataDir = process.env.DATA_DIR || join(process.cwd(), "data");
  const snapshotDir = join(dataDir, "snapshots");
  if (!existsSync(snapshotDir)) {
    mkdirSync(snapshotDir, { recursive: true });
  }

  /**
   * POST /api/snapshots/generate
   * 生成单页截图，返回截图 URL
   */
  fastify.post("/api/snapshots/generate", async (req: FastifyRequest, reply: FastifyReply) => {
    const body = req.body as GenerateSnapshotBody;
    const { projectId, pageId, code, schema, configData, width = 375, height = 812 } = body;

    if (!projectId || !pageId || !code) {
      return reply.status(400).send({
        success: false,
        error: { code: "INVALID_REQUEST", message: "缺少必填参数" },
      });
    }

    try {
      const renderer = getSnapshotRenderer();
      const pngBuffer = await renderer.render({
        code,
        schema,
        configData,
        width,
        height,
      });

      const pageDir = join(snapshotDir, projectId);
      if (!existsSync(pageDir)) {
        mkdirSync(pageDir, { recursive: true });
      }
      writeFileSync(join(pageDir, `${pageId}.png`), pngBuffer);

      return reply.send({
        success: true,
        data: {
          url: `/api/snapshots/file/${projectId}/${pageId}`,
        },
      });
    } catch (error) {
      fastify.log.error("截图生成失败:", error);
      return reply.status(500).send({
        success: false,
        error: { code: "SNAPSHOT_ERROR", message: "截图生成失败" },
      });
    }
  });

  /**
   * POST /api/snapshots/generate-batch
   * 批量生成截图（并发 5 个）
   */
  fastify.post("/api/snapshots/generate-batch", async (req: FastifyRequest, reply: FastifyReply) => {
    const body = req.body as GenerateBatchBody;
    const { projectId, pages } = body;

    if (!projectId || !pages?.length) {
      return reply.status(400).send({
        success: false,
        error: { code: "INVALID_REQUEST", message: "缺少必填参数" },
      });
    }

    const concurrency = 5;
    const results: Record<string, string> = {};
    const errors: string[] = [];

    const renderer = getSnapshotRenderer();

    for (let i = 0; i < pages.length; i += concurrency) {
      const batch = pages.slice(i, i + concurrency);
      await Promise.all(
        batch.map(async (page) => {
          try {
            const pngBuffer = await renderer.render({
              code: page.code,
              schema: page.schema,
              configData: page.configData,
              width: page.width || 375,
              height: page.height || 812,
            });

            const pageDir = join(snapshotDir, projectId);
            if (!existsSync(pageDir)) {
              mkdirSync(pageDir, { recursive: true });
            }
            writeFileSync(join(pageDir, `${page.pageId}.png`), pngBuffer);

            results[page.pageId] = `/api/snapshots/file/${projectId}/${page.pageId}`;
          } catch (error) {
            fastify.log.error(`截图生成失败 [${page.pageId}]:`, error);
            errors.push(page.pageId);
          }
        }),
      );
    }

    return reply.send({
      success: true,
      data: { urls: results, errors: errors.length > 0 ? errors : undefined },
    });
  });

  /**
   * GET /api/snapshots/file/:projectId/:pageId
   * 提供截图文件
   */
  fastify.get("/api/snapshots/file/:projectId/:pageId", async (req: FastifyRequest, reply: FastifyReply) => {
    const { projectId, pageId } = req.params as { projectId: string; pageId: string };
    const filePath = join(snapshotDir, projectId, `${pageId}.png`);

    if (!existsSync(filePath)) {
      return reply.status(404).send({
        success: false,
        error: { code: "SNAPSHOT_NOT_FOUND", message: "截图不存在" },
      });
    }

    const buffer = readFileSync(filePath);
    return reply
      .header("Content-Type", "image/png")
      .header("Cache-Control", "public, max-age=3600")
      .send(buffer);
  });
}
```

#### 4.3.2 Puppeteer 截图渲染器（agent-service）

```typescript
// packages/agent-service/src/utils/snapshot-renderer.ts

import puppeteer, { Browser } from "puppeteer";

interface RenderOptions {
  code: string;
  schema?: string;
  configData?: Record<string, unknown>;
  width: number;
  height: number;
}

let browserPromise: Promise<Browser> | null = null;

function getBrowser(): Promise<Browser> {
  if (!browserPromise) {
    browserPromise = puppeteer.launch({
      headless: true,
      args: [
        "--no-sandbox",
        "--disable-setuid-sandbox",
        "--disable-dev-shm-usage",
        "--disable-gpu",
      ],
    });
  }
  return browserPromise;
}

export function getSnapshotRenderer() {
  return {
    async render(options: RenderOptions): Promise<Buffer> {
      const { code, schema, configData = {}, width, height } = options;

      const html = buildSnapshotHtml({ code, schema, configData });

      const browser = await getBrowser();
      const page = await browser.newPage();

      try {
        await page.setViewport({ width, height, deviceScaleFactor: 1 });
        await page.setContent(html, {
          waitUntil: "networkidle0",
          timeout: 15000,
        });

        // 等待字体加载完成
        await page.evaluate(() => document.fonts.ready);

        // 额外等待 500ms 确保渲染稳定（CSS 动画/过渡）
        await new Promise((r) => setTimeout(r, 500));

        const buffer = (await page.screenshot({
          type: "png",
          fullPage: false,
          clip: { x: 0, y: 0, width, height },
        })) as Buffer;

        return buffer;
      } finally {
        await page.close();
      }
    },

    async destroy() {
      if (browserPromise) {
        const browser = await browserPromise;
        await browser.close();
        browserPromise = null;
      }
    },
  };
}

function buildSnapshotHtml(options: {
  code: string;
  schema?: string;
  configData: Record<string, unknown>;
}): string {
  const { code, schema, configData } = options;
  // 复用现有的 iframe HTML 生成逻辑，但移除 postMessage 监听（静态截图不需要）
  const initialConfig = JSON.stringify(configData || {});

  return `<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <style>
    * { margin: 0; padding: 0; box-sizing: border-box; }
    html, body, #root { width: 100%; height: 100%; overflow: hidden; }
  </style>
</head>
<body>
  <div id="root"></div>
  <script type="module">
    window.__DEMO_PROPS__ = ${initialConfig};

    const code = ${JSON.stringify(code)};
    const blob = new Blob([code], { type: 'application/javascript' });
    const url = URL.createObjectURL(blob);

    try {
      const mod = await import(url);
      const Component = mod.default;
      if (Component) {
        const root = document.getElementById('root');
        const { createRoot } = await import('https://esm.sh/react-dom@18/client');
        createRoot(root).render(React.createElement(Component));
      }
    } catch (err) {
      document.getElementById('root').innerHTML =
        '<div style="color:red;padding:20px;">' + err.message + '</div>';
    }
  </script>
  <script type="importmap">
  {
    "imports": {
      "react": "https://esm.sh/react@18",
      "react-dom": "https://esm.sh/react-dom@18"
    }
  }
  </script>
</body>
</html>`;
}
```

> **注意**：`buildSnapshotHtml` 需要根据项目中实际的 iframe 渲染模板调整。项目已有 `packages/shared/src/demo/iframe-template.ts` 中的 `generateIframeHtml` 函数，截图渲染器应复用其生成的 HTML（注入 code + configData）。

#### 4.3.3 CanvasViewport —— 平移/缩放容器

```typescript
// packages/shared/src/demo/CanvasViewport.tsx

"use client";

import React, { useState, useRef, useCallback, useEffect } from "react";
import { cn } from "./utils";
import type { CanvasViewport as CanvasViewportState } from "./types";

interface CanvasViewportProps {
  viewport: CanvasViewportState;
  onViewportChange: (viewport: CanvasViewportState) => void;
  editable?: boolean;
  onCanvasClick?: () => void;
  children?: React.ReactNode;
  className?: string;
}

export function CanvasViewport({
  viewport,
  onViewportChange,
  editable = true,
  onCanvasClick,
  children,
  className,
}: CanvasViewportProps) {
  const containerRef = useRef<HTMLDivElement>(null);
  const [isPanning, setIsPanning] = useState(false);
  const startPosRef = useRef({ x: 0, y: 0 });
  const viewportStartRef = useRef({ x: 0, y: 0 });

  const MIN_ZOOM = 0.05;
  const MAX_ZOOM = 3;
  const ZOOM_STEP = 1.1;

  const handleMouseDown = useCallback(
    (e: React.MouseEvent) => {
      // 左键或中键开始平移（排除右键）
      if (e.button !== 0 && e.button !== 1) return;
      // 点击在空白区域（非页面项）
      if (e.target === e.currentTarget || (e.target as HTMLElement).closest("[data-canvas-root]")) {
        setIsPanning(true);
        startPosRef.current = { x: e.clientX, y: e.clientY };
        viewportStartRef.current = { x: viewport.x, y: viewport.y };
        e.currentTarget.setPointerCapture(e.pointerId);
      }
    },
    [viewport.x, viewport.y],
  );

  const handleMouseMove = useCallback(
    (e: React.MouseEvent) => {
      if (!isPanning) return;
      const dx = e.clientX - startPosRef.current.x;
      const dy = e.clientY - startPosRef.current.y;
      onViewportChange({
        ...viewport,
        x: viewportStartRef.current.x + dx,
        y: viewportStartRef.current.y + dy,
      });
    },
    [isPanning, viewport, onViewportChange],
  );

  const handleMouseUp = useCallback(
    (e: React.MouseEvent) => {
      if (!isPanning) return;
      setIsPanning(false);
      (e.target as HTMLElement).releasePointerCapture?.(e.pointerId);
      // 如果几乎没移动，视为点击空白
      const dx = e.clientX - startPosRef.current.x;
      const dy = e.clientY - startPosRef.current.y;
      if (Math.abs(dx) < 3 && Math.abs(dy) < 3) {
        onCanvasClick?.();
      }
    },
    [isPanning, onCanvasClick],
  );

  const handleWheel = useCallback(
    (e: React.WheelEvent) => {
      e.preventDefault();
      const rect = containerRef.current?.getBoundingClientRect();
      if (!rect) return;

      const mouseX = e.clientX - rect.left;
      const mouseY = e.clientY - rect.top;

      const zoomFactor = e.deltaY > 0 ? 1 / ZOOM_STEP : ZOOM_STEP;
      const newZoom = Math.min(Math.max(viewport.zoom * zoomFactor, MIN_ZOOM), MAX_ZOOM);

      // 以鼠标位置为中心缩放
      const scale = newZoom / viewport.zoom;
      const newX = mouseX - (mouseX - viewport.x) * scale;
      const newY = mouseY - (mouseY - viewport.y) * scale;

      onViewportChange({ x: newX, y: newY, zoom: newZoom });
    },
    [viewport, onViewportChange],
  );

  return (
    <div
      ref={containerRef}
      className={cn(
        "w-full h-full overflow-hidden",
        isPanning ? "cursor-grabbing" : "cursor-grab",
        className,
      )}
      data-canvas-root="true"
      onMouseDown={handleMouseDown}
      onMouseMove={handleMouseMove}
      onMouseUp={handleMouseUp}
      onMouseLeave={handleMouseUp}
      onWheel={handleWheel}
    >
      <div
        style={{
          transform: `translate(${viewport.x}px, ${viewport.y}px) scale(${viewport.zoom})`,
          transformOrigin: "0 0",
          willChange: isPanning ? "transform" : "auto",
          width: 0,
          height: 0,
        }}
      >
        {children}
      </div>
    </div>
  );
}
```

#### 4.3.4 CanvasPageItem —— 图片或 Iframe 切换

```typescript
// packages/shared/src/demo/CanvasPageItem.tsx

"use client";

import React, { useState, useCallback, useRef, useEffect } from "react";
import { cn } from "./utils";
import { PreviewPanel } from "./PreviewPanel";
import type { CanvasPageLayout, CanvasPageData, PreviewSize } from "./types";

interface CanvasPageItemProps {
  page: CanvasPageData;
  layout: CanvasPageLayout;
  editable: boolean;
  isEditing?: boolean; // 当前是否处于 iframe 编辑模式
  snapshotUrl?: string;
  snapshotVersion?: number;
  sessionId?: string;
  onLayoutChange?: (pageId: string, layout: CanvasPageLayout) => void;
  onConfigEdit?: (pageId: string) => void;
  onDoubleClick?: (pageId: string) => void;
  onDragStart?: () => void;
  onDragEnd?: () => void;
}

export function CanvasPageItem({
  page,
  layout,
  editable,
  isEditing = false,
  snapshotUrl,
  snapshotVersion,
  sessionId,
  onLayoutChange,
  onConfigEdit,
  onDoubleClick,
  onDragStart,
  onDragEnd,
}: CanvasPageItemProps) {
  const [isDragging, setIsDragging] = useState(false);
  const [isHovering, setIsHovering] = useState(false);
  const startPosRef = useRef({ x: 0, y: 0 });
  const layoutStartRef = useRef({ x: 0, y: 0 });

  const handleDragStart = useCallback(
    (e: React.MouseEvent) => {
      if (!editable || isEditing) return;
      e.stopPropagation();
      setIsDragging(true);
      startPosRef.current = { x: e.clientX, y: e.clientY };
      layoutStartRef.current = { x: layout.x, y: layout.y };
      onDragStart?.();
    },
    [editable, isEditing, layout.x, layout.y, onDragStart],
  );

  const handleDragMove = useCallback(
    (e: React.MouseEvent) => {
      if (!isDragging) return;
      e.stopPropagation();
      const dx = e.clientX - startPosRef.current.x;
      const dy = e.clientY - startPosRef.current.y;
      onLayoutChange?.(page.id, {
        ...layout,
        x: layoutStartRef.current.x + dx,
        y: layoutStartRef.current.y + dy,
      });
    },
    [isDragging, page.id, layout, onLayoutChange],
  );

  const handleDragEnd = useCallback(() => {
    setIsDragging(false);
    onDragEnd?.();
  }, [onDragEnd]);

  const handleDoubleClick = useCallback(
    (e: React.MouseEvent) => {
      e.stopPropagation();
      onDoubleClick?.(page.id);
    },
    [page.id, onDoubleClick],
  );

  const handleConfigClick = useCallback(
    (e: React.MouseEvent) => {
      e.stopPropagation();
      if (editable && !isEditing) {
        onConfigEdit?.(page.id);
      }
    },
    [editable, isEditing, page.id, onConfigEdit],
  );

  const imageSrc = snapshotUrl
    ? `${process.env.NEXT_PUBLIC_AGENT_SERVICE_URL || ""}${snapshotUrl}?v=${snapshotVersion || 0}`
    : undefined;

  return (
    <div
      className={cn(
        "absolute transition-shadow duration-200",
        editable && !isEditing && "cursor-move",
        isDragging && "shadow-2xl opacity-90",
        isEditing && "ring-2 ring-blue-500",
      )}
      style={{
        left: layout.x,
        top: layout.y,
        width: layout.width,
        height: layout.height,
        zIndex: isEditing ? 10 : layout.zIndex ?? 0,
      }}
      onMouseDown={handleDragStart}
      onMouseMove={handleDragMove}
      onMouseUp={handleDragEnd}
      onMouseLeave={() => {
        handleDragEnd();
        setIsHovering(false);
      }}
      onMouseEnter={() => setIsHovering(true)}
      onDoubleClick={handleDoubleClick}
    >
      {isEditing ? (
        // iframe 模式（实时预览）
        <div className="w-full h-full rounded-lg overflow-hidden shadow-lg">
          <PreviewPanel
            code={page.code}
            sessionId={sessionId}
            demoId={page.id}
            configData={page.configData}
            previewSize={page.previewSize}
            snapshotVersion={snapshotVersion}
          />
        </div>
      ) : imageSrc ? (
        // 图片模式（快照）
        <>
          <img
            src={imageSrc}
            alt={page.name}
            className="w-full h-full object-cover rounded-lg shadow-md border border-border/50"
            draggable={false}
          />
          {/* 页面名称标签 */}
          <div className="absolute bottom-0 left-0 right-0 bg-gradient-to-t from-black/60 to-transparent p-2 rounded-b-lg">
            <span className="text-xs text-white font-medium truncate block">
              {page.name}
            </span>
          </div>
        </>
      ) : (
        // 加载中（骨架屏）
        <div className="w-full h-full bg-muted rounded-lg flex items-center justify-center border border-border/50">
          <div className="w-8 h-8 border-2 border-primary/30 border-t-primary rounded-full animate-spin" />
        </div>
      )}

      {/* 创作端：悬浮「修改配置」按钮 */}
      {editable && isHovering && !isEditing && (
        <button
          type="button"
          className="absolute top-2 right-2 bg-blue-500 hover:bg-blue-600 text-white text-xs px-2 py-1 rounded shadow transition-opacity"
          onClick={handleConfigClick}
        >
          修改配置
        </button>
      )}

      {/* 创作端拖动时的蓝色边框 */}
      {isDragging && (
        <div className="absolute inset-0 border-2 border-blue-500 rounded-lg pointer-events-none" />
      )}
    </div>
  );
}
```

#### 4.3.5 PreviewCanvas —— 主组件

```typescript
// packages/shared/src/demo/PreviewCanvas.tsx

"use client";

import React, { useState, useEffect, useRef, useCallback } from "react";
import { CanvasViewport } from "./CanvasViewport";
import { CanvasPageItem } from "./CanvasPageItem";
import { CanvasToolbar } from "./CanvasToolbar";
import { cn } from "./utils";
import type { PreviewCanvasProps, CanvasState, CanvasPageLayout } from "./types";

const DEFAULT_PAGE_SIZE = { width: 375, height: 812 };

function computeInitialLayout(
  pages: PreviewCanvasProps["pages"],
): Record<string, CanvasPageLayout> {
  const layout: Record<string, CanvasPageLayout> = {};
  const cols = 3;
  const gap = 40;

  pages.forEach((page, i) => {
    const col = i % cols;
    const row = Math.floor(i / cols);
    const w = DEFAULT_PAGE_SIZE.width;
    const h = DEFAULT_PAGE_SIZE.height;

    layout[page.id] = {
      x: col * (w + gap),
      y: row * (h + gap),
      width: w,
      height: h,
      zIndex: i,
    };
  });

  return layout;
}

export function PreviewCanvas({
  editable = false,
  sessionId,
  projectId,
  pages,
  canvasState: externalState,
  onCanvasStateChange,
  onPageConfigEdit,
  onCanvasClick,
  onPageDoubleClick,
  snapshotVersion = 0,
  className,
  editingPageId,
}: PreviewCanvasProps) {
  const [internalState, setInternalState] = useState<CanvasState>({
    viewport: { x: 40, y: 40, zoom: 1 },
    pages: computeInitialLayout(pages),
  });
  const [snapshotUrls, setSnapshotUrls] = useState<Record<string, string>>({});
  const [loading, setLoading] = useState(true);
  const [generatingPageId, setGeneratingPageId] = useState<string | null>(null);

  const canvasState = externalState || internalState;

  const updateState = useCallback(
    (updater: (prev: CanvasState) => CanvasState) => {
      const newState = updater(canvasState);
      if (externalState) {
        onCanvasStateChange(newState);
      } else {
        setInternalState(newState);
      }
    },
    [canvasState, externalState, onCanvasStateChange],
  );

  // 生成所有页面截图
  const generateAllSnapshots = useCallback(async () => {
    setLoading(true);
    try {
      const res = await fetch(
        `${process.env.NEXT_PUBLIC_AGENT_SERVICE_URL || ""}/api/snapshots/generate-batch`,
        {
          method: "POST",
          headers: { "Content-Type": "application/json" },
          body: JSON.stringify({
            projectId: projectId || "temp",
            pages: pages.map((p) => ({
              pageId: p.id,
              code: p.code || "",
              configData: p.configData,
              width: DEFAULT_PAGE_SIZE.width,
              height: DEFAULT_PAGE_SIZE.height,
            })),
          }),
        },
      );

      const data = await res.json();
      if (data.success) {
        setSnapshotUrls(data.data.urls || {});
      }
    } catch (error) {
      console.error("批量截图生成失败:", error);
    } finally {
      setLoading(false);
    }
  }, [pages, projectId]);

  // 重新生成单个页面截图
  const regenerateSnapshot = useCallback(
    async (pageId: string) => {
      const page = pages.find((p) => p.id === pageId);
      if (!page?.code) return;

      setGeneratingPageId(pageId);
      try {
        const res = await fetch(
          `${process.env.NEXT_PUBLIC_AGENT_SERVICE_URL || ""}/api/snapshots/generate`,
          {
            method: "POST",
            headers: { "Content-Type": "application/json" },
            body: JSON.stringify({
              projectId: projectId || "temp",
              pageId,
              code: page.code,
              configData: page.configData,
              width: DEFAULT_PAGE_SIZE.width,
              height: DEFAULT_PAGE_SIZE.height,
            }),
          },
        );

        const data = await res.json();
        if (data.success) {
          setSnapshotUrls((prev) => ({ ...prev, [pageId]: data.data.url }));
        }
      } catch (error) {
        console.error("重新截图失败:", error);
      } finally {
        setGeneratingPageId(null);
      }
    },
    [pages, projectId],
  );

  // 页面初始化时生成截图
  useEffect(() => {
    if (pages.length > 0) {
      generateAllSnapshots();
    }
  }, [pages.map((p) => p.id).join(","), snapshotVersion]);

  // 配置编辑结束时重新截图
  const handleCanvasClick = useCallback(() => {
    // 如果之前有编辑中的页面，重新生成其截图
    if (editingPageId) {
      regenerateSnapshot(editingPageId);
    }
    onCanvasClick?.();
  }, [editingPageId, regenerateSnapshot, onCanvasClick]);

  const handleLayoutChange = useCallback(
    (pageId: string, layout: CanvasPageLayout) => {
      updateState((prev) => ({
        ...prev,
        pages: { ...prev.pages, [pageId]: layout },
      }));
    },
    [updateState],
  );

  return (
    <div className={cn("w-full h-full relative overflow-hidden bg-muted/30", className)}>
      {editable && (
        <CanvasToolbar
          zoom={canvasState.viewport.zoom}
          onZoomChange={(zoom) =>
            updateState((prev) => ({
              ...prev,
              viewport: { ...prev.viewport, zoom },
            }))
          }
          onReset={() =>
            updateState((prev) => ({
              pages: computeInitialLayout(pages),
              viewport: { x: 40, y: 40, zoom: 0.5 },
            }))
          }
          loading={loading}
        />
      )}

      <CanvasViewport
        viewport={canvasState.viewport}
        onViewportChange={(viewport) =>
          updateState((prev) => ({ ...prev, viewport }))
        }
        editable={editable}
        onCanvasClick={handleCanvasClick}
      >
        {pages.map((page) => (
          <CanvasPageItem
            key={page.id}
            page={page}
            layout={canvasState.pages[page.id] || {
              x: 0,
              y: 0,
              width: DEFAULT_PAGE_SIZE.width,
              height: DEFAULT_PAGE_SIZE.height,
            }}
            editable={editable}
            isEditing={editingPageId === page.id}
            snapshotUrl={snapshotUrls[page.id]}
            snapshotVersion={
              generatingPageId === page.id ? undefined : snapshotVersion
            }
            sessionId={sessionId}
            onLayoutChange={handleLayoutChange}
            onConfigEdit={onPageConfigEdit}
            onDoubleClick={onPageDoubleClick}
          />
        ))}
      </CanvasViewport>

      {loading && (
        <div className="absolute inset-0 flex items-center justify-center bg-background/50">
          <div className="text-center">
            <div className="w-10 h-10 border-2 border-primary/30 border-t-primary rounded-full animate-spin mx-auto mb-2" />
            <p className="text-sm text-muted-foreground">正在生成页面快照...</p>
          </div>
        </div>
      )}
    </div>
  );
}
```

```typescript
import { debounce } from "./utils";

const SAVE_DELAY = 500;

export const saveCanvasLayout = debounce(
  async (
    sessionId: string,
```

## 五、风险与应对

| 风险 | 概率 | 影响 | 应对 |
|------|------|------|------|
| Puppeteer Chrome 未安装 | 低 | 高 | Docker 改用 `node:20-bookworm`（含 Chromium）；本地文档写明需手动安装 |
| 50 页截图耗时过长 | 中 | 低 | 并发 5，骨架屏加载提示，持续显示进度 |
| `<img>` 跨域问题 | 低 | 中 | 服务端设置 CORS headers 或使用 Next.js 代理路由 |
| iframe 在 canvas transform 中渲染异常 | 低 | 中 | iframe 不参与 canvas transform，独立定位 absolutely |
| 手势冲突 | 中 | 中 | `stopPropagation` + 事件优先级（页面拖拽 > 画布平移） |
