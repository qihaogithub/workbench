# AGENTS.md — docs/plans/进行中

> 本目录用于存放正在推进的任务计划、排查记录和优化方案。

## 写作边界

- 本目录文档属于任务追踪材料，不是长期项目知识库。
- 新增或更新本目录文档时，不需要同步 `docs/项目文档/INDEX.md`。
- 如果任务改变了产品行为、接口契约、架构边界或模块职责，再另行更新 `docs/项目文档/` 下对应模块文档。

## 必备内容

复杂任务文档至少包含：

- 背景
- 目标
- 范围
- 方案
- 任务清单
- 进度记录
- 验证方式
- 风险与待确认事项

任务清单应在实施过程中同步更新，不能只在最后一次性补写。

## 归档

- 任务完成后，按 `docs/plans/已完成/AGENTS.md` 的分类规则移动到已完成目录。
- 被新方案替代但仍有历史价值的文档，应在正文中标注状态和替代文档，而不是直接删除。
- 明确过期且无保留价值的文档，清理前先确认没有其他文档引用。
