## AI 对话窗口消息类型总览

本文档汇总了 AI 对话系统中所有消息类型的设计规范，包括角色分类、内容类型、工具状态、特殊消息和操作功能。

---

### 1. 角色分类

| 角色 | 说明 | 样式特征 |
| :--- | :--- | :--- |
| **user** | 用户发送的消息 | 右对齐，有背景色气泡 |
| **assistant** | AI 回复的消息 | 左对齐，无背景或浅色背景 |
| **system** | 系统消息（错误提示等） | 居中，特殊样式 |

---

### 2. 内容类型

| 内容类型 | 数据字段 | 展示方式 | 使用场景 |
| :--- | :--- | :--- | :--- |
| **文本消息** | `content: string` | Markdown 渲染，代码高亮 | 普通对话内容 |
| **思考过程** | `reasoning?: { content, duration }` | 折叠面板，等宽字体 | AI 推理过程 |
| **多步思考** | `reasonings?: Array<{ content, duration, timestamp }>` | 时间线展示多步思考 | 复杂推理场景 |
| **工具调用** | `tools?: Array<{ name, status, parameters, result }>` | 卡片展示，状态指示 | 文件读写、命令执行 |
| **图片** | `images?: Array<{ url, alt }>` | 网格布局展示 | 图片上传/生成 |
| **文件附件** | `files?: Array<{ name, url, size }>` | 文件卡片列表 | 文档上传 |
| **流式内容** | `isStreaming?: boolean` | 实时打字效果 | AI 正在生成回复 |
| **权限请求** | `PermissionRequest` | 弹窗确认 | 敏感操作确认 |

---

### 3. 工具调用状态

| 状态 | 说明 | UI 展示 |
| :--- | :--- | :--- |
| **running** | 正在执行 | 加载动画 + 进度条 |
| **completed** | 执行成功 | 绿色勾选 + 结果展示 |
| **error** | 执行失败 | 红色错误提示 |
| **awaiting-approval** | 等待用户确认 | 权限请求弹窗 |

---

### 4. 特殊消息类型

| 类型 | 说明 | 触发条件 |
| :--- | :--- | :--- |
| **错误消息** | WebSocket/HTTP 请求失败 | 网络错误或 Agent 服务异常 |
| **加载中** | AI 正在思考但未开始输出 | 等待 AI 响应的骨架屏 |
| **权限请求** | 需要用户确认的操作 | 文件修改、命令执行等敏感操作 |

---

### 5. 消息操作功能

| 操作 | 适用角色 | 功能说明 |
| :--- | :--- | :--- |
| **复制** | assistant | 复制消息内容到剪贴板 |
| **点赞** | assistant | 反馈消息有用 |
| **点踩** | assistant | 反馈消息无用 |
| **重新生成** | assistant | 重新请求 AI 生成回复 |
| **编辑** | user | 修改已发送的消息 |
