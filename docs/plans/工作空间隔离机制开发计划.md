## 一、概述

### 1.1 目标

在独立 Agent 服务层实现与 AionUI 相同的工作空间隔离机制，确保：

1. **每个 Session 绑定独立工作空间**
2. **Agent 只能操作自己工作空间内的文件**
3. **支持用户指定工作空间和系统临时工作空间**
4. **支持工作空间切换**
5. **支持文件变更追踪和回滚**

#### AionUI 工作空间创建

```typescript
// AionUI: src/process/utils/initAgent.ts
const buildWorkspaceWidthFiles = async (
  defaultWorkspaceName: string,
  workspace?: string,
  providedCustomWorkspace?: boolean
) => {
  const customWorkspace = providedCustomWorkspace !== undefined 
    ? providedCustomWorkspace 
    : !!workspace;

  if (!workspace) {
    // 创建临时工作空间
    const tempPath = getSystemDir().workDir;
    workspace = path.join(tempPath, defaultWorkspaceName);
    await fs.mkdir(workspace, { recursive: true });
  } else {
    // 规范化用户指定路径
    workspace = path.resolve(workspace);
  }

  return { workspace, customWorkspace };
};
```

```typescript
/**
 * 工作空间类型
 */
export interface WorkspaceInfo {
  /** 工作空间路径 */
  path: string;
  /** 是否为用户指定工作空间 */
  customWorkspace: boolean;
  /** 工作空间类型 */
  type: 'user' | 'temp';
  /** 创建时间 */
  createdAt: number;
}

/**
 * 工作空间创建选项
 */
export interface CreateWorkspaceOptions {
  /** 后端类型，用于生成临时目录名 */
  backend: string;
  /** 用户指定的工作空间路径（可选） */
  workspace?: string;
  /** 是否为用户指定工作空间（可选，由 workspace 参数自动推断） */
  customWorkspace?: boolean;
}

/**
 * 文件变更信息
 */
export interface FileChange {
  /** 文件路径（相对于工作空间） */
  path: string;
  /** 操作类型 */
  operation: 'create' | 'modify' | 'delete';
  /** 变更状态 */
  status: 'staged' | 'unstaged';
}

/**
 * 工作空间快照信息
 */
export interface SnapshotInfo {
  /** 快照模式 */
  mode: 'git-repo' | 'snapshot';
  /** 当前分支（git-repo 模式） */
  branch: string | null;
}

/**
 * 文件变更对比结果
 */
export interface CompareResult {
  /** 已暂存的变更 */
  staged: FileChange[];
  /** 未暂存的变更 */
  unstaged: FileChange[];
}
```

## 六、验收标准

### 功能验收

| 功能 | 验收标准 |
|:-----|:---------|
| **临时工作空间** | 创建 Session 时不指定 workingDir，自动创建临时目录 |
| **用户指定工作空间** | 创建 Session 时指定 workingDir，使用指定目录 |
| **工作空间切换** | 调用 API 切换工作空间，Session 绑定新目录 |
| **路径隔离** | Agent 无法访问工作空间外的文件 |
| **变更检测** | 能检测文件的新增、修改、删除 |
| **变更回滚** | 能回滚到变更前的状态 |
| **临时空间清理** | Session 销毁时清理临时工作空间 |

### 安全验收

| 安全项 | 验收标准 |
|:-------|:---------|
| **路径遍历防护** | 无法通过 `../` 访问工作空间外的文件 |
| **符号链接防护** | 符号链接指向工作空间外时拒绝访问 |
| **权限隔离** | 不同 Session 的工作空间相互隔离 |

### 性能验收

| 性能项 | 验收标准 |
|:-------|:---------|
| **快照初始化** | < 1s（小型项目） |
| **变更检测** | < 500ms |
| **路径校验** | < 10ms |

## 七、风险与应对

| 风险 | 影响 | 应对措施 |
|:-----|:-----|:---------|
| **Git 操作兼容性** | 不同平台 Git 行为可能不同 | 使用 isomorphic-git 或充分测试 |
| **大文件变更检测** | 大型项目变更检测慢 | 增量检测、缓存机制 |
| **并发访问** | 多个 Agent 同时操作同一工作空间 | 加锁机制、冲突检测 |
