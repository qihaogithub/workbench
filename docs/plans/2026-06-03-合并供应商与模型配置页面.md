# 合并供应商与模型配置页面 实施计划

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**目标：** 将「AI 后端供应商」和「模型配置」两个独立页面合并为一个带标签页的「AI 模型管理」页面，简化保存流程，自动加载模型，减少用户操作步骤。

**架构：** 新建 `/admin/models` 页面，通过 Tabs 组件切换供应商管理和模型白名单两个视图。旧页面添加重定向。侧边栏合并为一个入口。

**技术栈：** Next.js 14 App Router, React, Tailwind CSS, shadcn/ui (Tabs, Button, Switch, Badge, Input 等), DnD Kit (拖拽排序)

**供应商 Tab 的 UX 简化：**

3. 每个模型行显示所属供应商的彩色 Badge
4. 保留现有的拖拽排序、开关启用、多模态标记等全部功能

**Step 1: 创建页面骨架和标签切换逻辑**

使用 shadcn/ui 的 Tabs 组件，默认显示供应商管理 Tab。通过 URL 查询参数 `?tab=` 控制默认 Tab。

```tsx
"use client";

import { useState, useCallback } from "react";
import { useSearchParams, useRouter } from "next/navigation";
import { Tabs, TabsContent, TabsList, TabsTrigger } from "@/components/ui/tabs";
import { Server, ListFilter } from "lucide-react";

export default function ModelsPage() {
  const searchParams = useSearchParams();
  const router = useRouter();
  const defaultTab = searchParams.get("tab") || "providers";
  const [activeTab, setActiveTab] = useState(defaultTab);

  const handleTabChange = useCallback((value: string) => {
    setActiveTab(value);
    router.replace(`/admin/models?tab=${value}`, { scroll: false });
  }, [router]);

  return (
    <Tabs value={activeTab} onValueChange={handleTabChange} className="space-y-6">
      <div className="flex items-center justify-between">
        <div>
          <h2 className="text-2xl font-bold text-neutral-50">AI 模型管理</h2>
          <p className="text-neutral-400 mt-1 text-sm">管理后端供应商和前端模型白名单</p>
        </div>
      </div>
      <TabsList className="bg-neutral-800 border border-neutral-700">
        <TabsTrigger value="providers" className="data-[state=active]:bg-neutral-700">
          <Server className="h-4 w-4 mr-2" />后端供应商
        </TabsTrigger>
        <TabsTrigger value="config" className="data-[state=active]:bg-neutral-700">
          <ListFilter className="h-4 w-4 mr-2" />模型白名单
        </TabsTrigger>
      </TabsList>
      <TabsContent value="providers">
        <SuppliersTab />
      </TabsContent>
      <TabsContent value="config">
        <ModelConfigTab />
      </TabsContent>
    </Tabs>
  );
}
```

- 添加推送状态的醒目展示（成功/失败+重试）
